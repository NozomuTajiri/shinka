- その他（ツール、イベント等）: ¥200,000（10%）

#### 顧客生涯価値（LTV）

**前提条件**:
- 平均単価: ¥15,000,000/年
- 粗利率: 85%（SaaS）
- チャーンレート: 5%/年（年間継続率95%）
- 平均顧客ライフタイム: 1 / 0.05 = 20年

**LTV計算**:
```
LTV = 平均単価 × 粗利率 × 平均ライフタイム
    = ¥15,000,000 × 0.85 × 20年
    = ¥255,000,000
```

**実質LTV（割引率考慮）**:
- 割引率（WACC）: 10%
- 実質平均ライフタイム: 1 / (0.05 + 0.10) = 6.67年
- **実質LTV**: ¥15,000,000 × 0.85 × 6.67 = **¥85,027,500**

**保守的LTV（5年）**: ¥15,000,000 × 0.85 × 5年 = **¥63,750,000**

#### ユニットエコノミクス指標

| 指標 | 値 | 業界ベンチマーク | 評価 |
|------|-----|----------------|------|
| **LTV（5年）** | ¥63,750,000 | - | - |
| **CAC** | ¥2,000,000 | - | - |
| **LTV/CAC** | **31.9** | 3.0以上が健全 | 優秀 |
| **CAC Payback Period** | **1.6ヶ月** | 12ヶ月以内が健全 | 優秀 |
| **Gross Margin** | 85% | 70%以上 | 優秀 |

**CAC Payback Period計算**:
```
Payback = CAC / (月次MRR × Gross Margin)
        = ¥2,000,000 / (¥1,250,000 × 0.85)
        = 1.88ヶ月
```
