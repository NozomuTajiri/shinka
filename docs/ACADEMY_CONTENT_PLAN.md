## エグゼクティブサマリー

### ビジョン

**「AI時代の付加価値創造を、すべてのビジネスパーソンに」**

価値主義AI Academyは、株式会社カクシンの「付加価値のつくりかた」メソッドとMiyabi AI自律開発フレームワークを融合した、日本初のAI価値創造教育プラットフォームです。

### 事業概要

- **事業領域**: AI×価値主義経営の教育・認定プラットフォーム
- **主要顧客**: エンタープライズ企業の経営者・エンジニア・営業職
- **収益モデル**: コース販売 + 企業ライセンス + 認定制度
- **目標時価総額**: 50億〜100億円（2026年）

### 財務ハイライト（2026年目標）

| 指標 | 目標値 |
|------|--------|
| ARR（年間経常収益） | 20億円 |
| 受講者数（累計） | 5,000名 |
| 企業ライセンス契約 | 100社 |
| 認定資格保有者 | 1,500名 |
| 粗利率 | 70% |

## 3. 認定制度設計

### 3.1 認定体系

```
【3段階認定制度】

Level 1: Miyabi認定アソシエイト
 ├─ コース修了条件: いずれか1コース修了
 ├─ 有効期限: 2年間
 └─ 更新条件: 継続教育プログラム受講

Level 2: Miyabi認定プロフェッショナル
 ├─ 条件: 2コース以上修了 + 実務経験6ヶ月
 ├─ 試験: 実技試験（プロジェクト提出）
 ├─ 有効期限: 3年間
 └─ 更新条件: 継続教育プログラム + 事例発表

Level 3: Miyabi認定マスター
 ├─ 条件: 全4コース修了 + 実務経験2年 + 論文審査
 ├─ 試験: 面接審査（田尻望含む審査委員会）
 ├─ 有効期限: 5年間
 └─ 特典: Miyabiコミュニティ運営権、講師資格
```

### 3.2 認定試験内容

#### Level 1: Miyabi認定アソシエイト

**試験なし（コース修了で自動付与）**

- コース内評価が基準点以上
- 出席率80%以上
- 課題提出100%

**認定証**: デジタル証明書（LinkedIn連携可能）

#### Level 2: Miyabi認定プロフェッショナル

**実技試験**:
1. **プロジェクト提出**: 実務でのMiyabi活用事例
2. **成果報告**: ROI、生産性向上率等の定量データ
3. **プレゼン**: 審査委員へのプレゼンテーション（30分）

**審査基準**:
- プロジェクトの独自性（30点）
- ビジネス成果（定量データ）（40点）
- プレゼンテーション（30点）

合計70点以上で合格

**認定証**: 紙証書 + デジタル証明書 + バッジ（物理・デジタル）

#### Level 3: Miyabi認定マスター

**論文審査**:
- テーマ: 「価値主義AI経営の実践と考察」（10,000字以上）
- 査読: 田尻望 + 外部有識者2名

**面接審査**:
- 審査委員: 田尻望、CTO、外部有識者2名
- 内容: 論文内容深堀り + ビジョンプレゼン（60分）

**審査基準**:
- 論文の学術的価値（40点）
- 実務での革新性（30点）
- 業界への貢献度（30点）

合計80点以上で合格

**特典**:
- 金色の認定証（田尻望サイン入り）
- Miyabiコミュニティ運営権
- Academy講師資格
- Forbes寄稿推薦

### 3.3 継続教育プログラム（CPD: Continuing Professional Development）

**年間要件**:
- Level 1: 10CPDポイント/年
- Level 2: 20CPDポイント/年
- Level 3: 30CPDポイント/年
