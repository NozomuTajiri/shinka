# 第20章 価値の連鎖

価値は伝染する

---

## 1. 一年後の大和精密工業

「山田本部長、おはようございます」

エレベーターホールで、若手社員たちが明るく挨拶してきた。

「おはよう」

山田浩一は、笑顔で応えた。

あれから一年。山田は営業本部長として、会社全体の営業戦略を統括する立場になっていた。

そして、大和精密工業は劇的に変わっていた。

エントランスを抜けると、壁一面に掲示された「価値創造ボード」が目に入った。

```
【今月お客様に届けた価値】

営業部: 12社に「生産性向上」の価値を提供
技術部: 5つの新製品で「リスク回避」を実現
製造部: 納期短縮で「時間節約」を達成
管理部: 業務効率化で全社の「働き方改革」を推進

合計: 127の価値創造を実現
```

訪問件数でも、提案数でもない。「届けた価値」が、全社の共通言語になっていた。

オフィスに入ると、佐藤課長が待っていた。

「本部長、報告があります」

「どうした？」

「先月、カワシマ精密工業に提供したトレーサビリティシステムですが——」

佐藤は、興奮した様子で続けた。

「川島社長が、業界団体の会合で、うちの取り組みを紹介してくださったそうです」

「ほう」

「それで、業界大手のサンエイ工業から問い合わせが来ました。『同じような価値提案をしてほしい』と」

山田は、深く頷いた。

「価値は、伝染するんだな」

「はい。まさに、そう感じます」

---

## 2. 業界団体からの招待

数日後、山田のもとに一通のメールが届いた。

差出人は、日本精密機械工業会。業界最大の団体だ。

```
件名: 講演依頼のお願い

山田浩一様

拝啓
時下ますますご清祥のこととお慶び申し上げます。

貴社の「価値起点経営」の取り組みが、業界内で大きな話題となっております。

つきましては、来月開催される当工業会の年次大会にて、
基調講演をお願いできないかと存じます。

テーマ: 「お客様に価値を届ける経営への転換」

ご検討のほど、よろしくお願い申し上げます。

敬具
```

山田は、画面を見つめた。

一年前の自分なら、信じられなかっただろう。

自分が業界団体で講演する日が来るなんて。

しかし、今は違う。

この学びを、業界全体に広げるチャンスだ。

山田は、返信を書き始めた。

---

## 3. 取引先の変化

その週、山田は久しぶりに中村製作所を訪れた。

十年来の取引先。一年前、価格競争で競合に負けそうになった顧客だ。

「山田さん!」

中村社長が、笑顔で出迎えた。

「お久しぶりです」

「いやあ、来てもらって嬉しいですよ」

応接室に通されると、中村社長は興奮した様子で話し始めた。

「山田さんのおかげで、うちも変わったんです」

「と、言いますと?」

「あのトレーサビリティシステムを導入してから、不良品の流出がゼロになりました。それで——」

中村社長は、資料を取り出した。

「大手自動車メーカーから、新規取引の申し込みがあったんです」

「それは、素晴らしい」

「でも、それだけじゃないんです」

中村社長は続けた。

「山田さんから教わった『価値起点』の考え方を、うちの営業にも導入したんです」

「え......?」

「お客様に『どんな価値を届けるか』を考えて提案するようにしたら、受注率が三割も上がりました」

山田は、驚いた。

「中村社長......」

「山田さんが届けてくれたのは、システムだけじゃなかった。『考え方』そのものだったんです」

中村社長は、深く頭を下げた。

「本当に、ありがとうございました」

---

## 4. パートナー企業への波及

翌月、山田は協力会社である東洋エンジニアリングの社長、田中と会食していた。

「山田さん、実は相談があるんです」

田中社長は、真剣な表情で切り出した。

「うちの営業チームが、伸び悩んでいましてね」

「どんな状況ですか?」

「みんな一生懸命働いているんです。でも、成果が出ない。山田さんの会社も、一年前は同じような状況だったと聞きました」

「ええ、まさにその通りでした」

「それで......よければ、うちの営業チームにも、『価値起点』の考え方を教えていただけませんか?」

山田は、少し考えた。

「田中社長、それは構いませんが——」

「はい」

「教えるだけでは、変わりません」

「......?」

山田は、グラスを置いた。

「大事なのは、トップである田中社長自身が、『価値起点』を実践することです」

「私が......ですか」

「そうです。組織の変革は、トップが変わらないと始まりません」

田中社長は、深く頷いた。

「わかりました。まず、私が学びます」

---

## 5. 社内の進化

大和精密工業の社内では、さらなる進化が起きていた。

製造部長の佐々木が、山田を訪ねてきた。

「本部長、報告です」

「どうした?」

「製造部で、新しい取り組みを始めました」

佐々木は、資料を広げた。

```
【製造部の価値創造プロジェクト】

従来: 「高品質な製品を作る」
↓
新方針: 「お客様の課題を解決する製品を作る」

具体的施策:
1. お客様の工場を訪問し、現場の課題をヒアリング
2. 課題解決に最適な製品仕様を提案
3. 試作品を現場でテストし、フィードバックを反映
```

「製造部が......お客様訪問を?」

山田は驚いた。

「はい。最初は抵抗がありました。『製造は工場で製品を作るのが仕事だ』と」

「でも、変わったんですね」

「ええ。実際にお客様の現場を見たら、みんなの目の色が変わったんです」

佐々木は続けた。

「『自分たちが作っている製品が、こんな風に使われているのか』『ここを改善すれば、もっと価値を届けられる』——現場の職人たちが、自発的に提案するようになりました」

「それは......素晴らしい」

「もう、製造部は『作る人』じゃありません。『価値を創造する人』です」

山田は、胸が熱くなった。

価値起点の考え方が、組織の隅々まで浸透している。

---

## 6. 業界大会での基調講演

そして、日本精密機械工業会の年次大会の日が来た。

会場には、業界の主要企業から五百名以上が集まっていた。

壇上に立った山田は、深呼吸をした。

「皆さん、こんにちは。大和精密工業の山田浩一と申します」

スクリーンに、一年半前の営業部の写真が映し出された。

疲れ切った表情のメンバーたち。灰色のオフィス。

「これは、一年半前の私たちです」

山田は、正直に語り始めた。

「当時、私たちは疲弊していました。毎日夜遅くまで働き、訪問件数を増やし、必死に努力していました」

「でも、受注は減り続けていました」

会場が、静まり返った。

「そんなとき、ある顧問から教わったんです。『努力の方向が間違っている』と」

山田は、スライドを進めた。

```
【私たちが学んだこと】

努力の量 ではなく
努力の質

訪問件数 ではなく
届けた価値

働いた時間 ではなく
解決した課題
```

「私たちは、『お客様起点』の経営に転換しました」

山田は、この一年半の取り組みを説明した。

六つの価値の理論。
お客様の本当の課題を見つける方法。
価値を見える化する提案書。
そして、組織全体での価値創造文化の醸成。

「結果、受注額は前年比で二倍になりました」

どよめきが起きた。

「でも、それ以上に大切なことがあります」

山田は、最新の社内写真を映した。

生き生きとした表情のメンバーたち。明るいオフィス。

「社員たちが、誇りを持って働けるようになったことです」

山田は続けた。

「お客様に価値を届ける。その喜びを知った社員は、もう疲弊しません」

「なぜなら、自分の仕事が、誰かの役に立っていると実感できるからです」

会場全体が、深く頷いていた。

「そして、この変化は——」

山田は、力を込めて言った。

「お客様にも、パートナー企業にも、広がっています」

スクリーンに、中村製作所、タナカ工業、カワシマ精密工業の事例が映し出された。

「私たちが価値を届けたお客様は、その価値を、さらに次のお客様に届けています」

「これが、『価値の連鎖』です」

山田は、最後のスライドを映した。

```
【価値の連鎖】

1. 私たちがお客様に価値を届ける
   ↓
2. お客様がその価値を実感し、成長する
   ↓
3. お客様が次のお客様に価値を届ける
   ↓
4. 業界全体が価値起点になる
   ↓
5. 社会全体に価値が広がる
```

「皆さん」

山田は、会場を見渡した。

「価値は、伝染します」

「一社が変われば、取引先が変わる。取引先が変われば、業界が変わる。業界が変われば、社会が変わる」

「今日、この会場にいる皆さんが、『価値起点』の経営を始めたら——」

山田は、声を張り上げた。

「日本の製造業全体が、変わります」

会場全体が、総立ちで拍手した。

---

## 7. 講演後の反響

講演が終わると、山田のもとに次々と企業の経営者が訪れた。

「ぜひ、うちでも取り組みたい」
「詳しく話を聞かせてほしい」
「コンサルティングをお願いできないか」

名刺交換だけで一時間以上かかった。

休憩室に戻ると、宮本顧問が待っていた。

「先生!」

「素晴らしい講演だったよ、山田君」

「先生のおかげです」

「いや」宮本は首を振った。「君が実践し、証明したんだ」

宮本は、窓の外を見た。

「山田君、君は気づいているか?」

「何をでしょうか」

「君が今日、この会場に植えた種がどれだけ大きく育つか」

山田は、黙って聞いた。

「今日ここにいた五百人が、それぞれの会社で『価値起点』を実践したら?」

「......」

「その五百社が、それぞれ百社の取引先に影響を与えたら?」

宮本は、山田の目を見た。

「五万社だ。日本の製造業の、相当な部分をカバーできる」

「五万社......」

「そして、それぞれの会社で働く人たちが、誇りを持って働けるようになったら——」

宮本は、優しく微笑んだ。

「何十万人もの人生が、変わるんだよ」

山田は、言葉を失った。

自分がやってきたことが、そんなに大きな影響を持つなんて。

「山田君、これが『価値の連鎖』だ」

---

## 8. 新しいミッション

数週間後、山田は社長室に呼ばれた。

「山田君、座ってくれ」

白石社長が、穏やかな表情で言った。

「業界大会での講演、素晴らしかったそうだね」

「ありがとうございます」

「実は、専務と相談してね」

白石社長は、一枚の紙を差し出した。

```
【新設: 価値創造推進室】

室長: 山田浩一

ミッション:
1. 社内の価値創造文化をさらに深化させる
2. 取引先・パートナー企業への価値創造支援
3. 業界全体への価値創造の普及活動
```

「これは......」

「君には、営業本部長を続けながら、この新しい部門も統括してほしい」

白石社長は続けた。

「大和精密工業の使命は、もう単に利益を上げることだけじゃない」

「社長......」

「業界全体を、お客様起点に変えていく。それが、私たちの新しい使命だ」

白石社長は、窓の外を見た。

「山田君、君が始めたこの変革を、業界全体に広げてくれ」

山田は、深く頷いた。

「はい。必ずやり遂げます」

---

## 9. 取引先への支援プログラム

価値創造推進室が発足すると、山田はまず取引先への支援プログラムを立ち上げた。

```
【価値創造パートナーシッププログラム】

対象: 大和精密工業の取引先・協力会社

内容:
1. 価値起点経営の基礎講座（全3回）
2. 現場訪問による価値発見ワークショップ
3. 価値提案書作成支援
4. 継続的なフォローアップ

費用: 無料（パートナーシップの一環として）
```

最初のセミナーには、三十社以上が参加した。

「皆さん、こんにちは」

山田は、ホワイトボードの前に立った。

「今日から三回にわたって、『価値起点経営』についてお話しします」

「でも、これは講義ではありません」

山田は、参加者を見渡した。

「皆さん自身が、『価値とは何か』を発見する場です」

そして、山田は一年半前に宮本から教わったことを、そのまま伝え始めた。

六つの価値の理論。
お客様の本当の課題を見つける質問法。
価値を見える化する方法。

参加者たちは、真剣にメモを取っていた。

---

## 10. 小さな会社の大きな変化

プログラムが進むにつれ、参加企業から次々と成功事例が報告されるようになった。

従業員十名の小さな部品メーカー、ハヤシ工業の林社長が報告に来た。

「山田さん、受注が三倍になりました!」

「それは素晴らしい。どんな取り組みをされたんですか?」

「セミナーで教わった『六つの価値』を使って、既存のお客様を全部回ったんです」

林社長は興奮した様子で続けた。

「そしたら、みんな困ってることがあったんです。『納期がギリギリで困る』『品質のバラツキが不安』『在庫管理が大変』——」

「それで?」

「うちの強みと照らし合わせたら、全部解決できることばかりでした。短納期対応、安定した品質、小ロット対応——」

林社長は、目を輝かせた。

「今までは『部品を売る』だけでした。でも、これからは『お客様の課題を解決する』んです」

「素晴らしい気づきですね」

「山田さんのおかげです。小さな会社でも、価値を届けられるんだって、自信が持てました」

大和精密工業の取り組みは、やがて業界メディアでも取り上げられるようになった。

「日本機械工業新聞」の一面に、大きく記事が掲載された。

```
【特集】価値起点経営で業界を変える

大和精密工業の挑戦
～受注2倍、でも本当の成果は「人の変化」～

中堅部品メーカーの大和精密工業が、
独自の「価値起点経営」で業績V字回復を実現。
さらに、その手法を取引先に無償で伝授し、
業界全体の底上げを目指している。

「利益は後からついてくる。まず、お客様に
価値を届けることに全力を尽くす」
（山田浩一・価値創造推進室長）
```

記事が出ると、全国から問い合わせが殺到した。

「詳しく話を聞きたい」
「うちでもセミナーを開いてほしい」
「コンサルティングをお願いしたい」

山田は、嬉しい悲鳴を上げていた。

「山田室長、お時間をいただけますでしょうか」

「当社は、従業員五千名を超える企業です」

岡田専務は、真剣な表情で言った。

「しかし、正直に申し上げると、組織が硬直化しています」

「......」

「社員は一生懸命働いていますが、疲弊している。お客様からのクレームも増えている」

岡田専務は、資料を開いた。

「大和精密工業の取り組みを知って、驚きました。中堅企業が、ここまで組織を変革できるなんて」

「ありがとうございます」

「それで、お願いがあります」

岡田専務は、深く頭を下げた。

「当社の組織変革を、支援していただけませんか」

山田は、一瞬戸惑った。

業界最大手の、五千人の組織を変える?

しかし、すぐに決意した。

「わかりました。全力でサポートします」

「山田様の取り組みを、国の施策として全国展開したいと考えています」

「日本の製造業全体が、価値起点に変われば——」

「パパ、かっこいい」

## 15. 価値創造サミット

そして、年の瀬。

大和精密工業の主催で、「価値創造サミット」が開催された。

会場には、全国から三百社以上の企業が集まった。

壇上には、山田、宮本顧問、そしてサンエイ工業の岡田専務の姿があった。

「皆さん、本日はお集まりいただき、ありがとうございます」

山田が、開会の挨拶をした。

「このサミットの目的は、『価値の連鎖』を加速させることです」

スクリーンに、この一年半の成果が映し出された。

```
【価値創造の広がり】

- 直接支援企業: 87社
- プログラム参加企業: 234社
```

## 本章のキーメッセージ

**価値は伝染する——一人から組織へ、組織から業界へ、業界から社会へ**

山田が学んだ「価値起点」の考え方は、もはや一企業の取り組みではなくなっていた。

```
【価値の連鎖 5つの段階】

第1段階: 個人の変化
└─ 山田が「価値起点」を実践

第2段階: チームの変化
└─ 営業部全体が変わる

第3段階: 組織の変化
└─ 全社が「価値創造」文化に

第4段階: 取引先・業界への波及
└─ パートナー企業、競合他社にも広がる

第5段階: 社会全体への影響
└─ 異業種、他地域、国全体へ
```

価値は、ウイルスのように伝染する。

良い意味で。

一人が価値を届けると、その人が変わる。
その人が変わると、周りの人が変わる。
周りの人が変わると、組織が変わる。
組織が変わると、取引先が変わる。
取引先が変わると、業界が変わる。
業界が変わると、社会が変わる。

山田の旅は、「報われない努力」から始まった。

しかし、今や彼の影響は、何十万人もの働く人々に届いている。

そして、この連鎖は、まだ終わらない。

今、この物語を読んでいるあなたも——

価値の連鎖の、次の担い手になれる。
