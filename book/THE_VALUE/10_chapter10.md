# 第10章 相手の立場で考える

6つの価値で経営課題を分析する

---

## 1. 新たなチャレンジ

月曜日の朝、山田は営業部の会議室に呼ばれた。

田村専務と林部長が既に席に着いており、テーブルの上には分厚い資料が置かれていた。

「山田君、座りたまえ」

田村専務の声には、いつもと違う期待感が滲んでいた。

「実は、大型の案件が舞い込んだ」

林部長が資料を開いた。

「日東自動車部品。従業員500名、年商150億円。自動車部品メーカーとしては中堅だが、大手メーカーへの供給実績が豊富な優良企業だ」

山田は資料に目を通した。確かに業界では名の知れた企業だ。

「先方の購買部長から、直接当社への問い合わせがあった」田村専務が続けた。「ナカムラ工業の中村社長から推薦されたらしい」

「中村社長が...」

山田の胸が熱くなった。あの成功事例が、次の扉を開いてくれたのだ。

「ただし」林部長が真剣な表情になった。「これは今までとはレベルが違う。相手は大企業だ。意思決定も複雑で、関係者も多い。中途半端な提案では通用しない」

「分かっています」

山田は背筋を伸ばした。

「初回の面談は来週の水曜日、午後2時。場所は先方の本社だ」田村専務が手帳を確認した。「準備期間は1週間。この案件、君に任せたい」

「ありがとうございます。必ず成功させます」

## 2. 宮本への相談

その日の夕方、山田は宮本の自宅を訪れた。

「先生、大型案件の相談があります」

宮本はコーヒーを淹れながら、山田の話を聞いた。

「日東自動車部品か。良い会社だな」

「はい。でも、正直プレッシャーを感じています。今までとは規模が違いすぎて」

「当然だ。大企業の攻略は、中小企業とは戦略が違う」

宮本は書斎の本棚から、一冊のノートを取り出した。表紙には「付加価値のつくりかた - 実践編」と書かれている。

「山田君、君はこれまで『価値とは何か』を学んできた。そして、個別のニーズに応える方法も身につけた」

「はい」

「だが、大企業を相手にするには、もう一段階上の視点が必要だ」

宮本はノートを開いた。そこには、整然とした図が描かれていた。

```
【6つの価値フレームワーク】

1. 機能的価値（Functional Value）
   └─ 商品・サービスの基本機能

2. 経済的価値（Economic Value）
   └─ コスト削減・利益向上

3. 感情的価値（Emotional Value）
   └─ 安心感・誇り・喜び

4. 社会的価値（Social Value）
   └─ 社会貢献・CSR

5. 体験的価値（Experiential Value）
   └─ 使用体験・顧客体験

6. 象徴的価値（Symbolic Value）
   └─ ブランド・ステータス
```

「これが、顧客が感じる価値の6つの側面だ」

山田は食い入るように図を見つめた。

「今まで君が扱ってきた案件は、主に『機能的価値』と『経済的価値』だった。ナカムラ工業の事例では、『感情的価値』—従業員の家族との時間—にも踏み込んだ」

「はい」

「だが、大企業を攻略するには、この6つすべての視点で顧客を分析する必要がある」

宮本は新しいページを開いた。

「これから、この6つの価値を使って、日東自動車部品を徹底的に分析する練習をしよう」

## 3. 情報収集の開始

翌日、山田は日東自動車部品について、あらゆる情報を集め始めた。

企業ホームページ、IR資料、業界誌の記事、SNS、求人サイト。デジタルで手に入る情報をすべて収集した。

さらに、業界関係者への電話も欠かさなかった。

「もしもし、ナカムラ工業の中村社長ですか？山田です」

「おお、山田さん。日東自動車部品の件、聞いたよ」

「はい。実は、先方について教えていただきたくて」

中村社長は快く情報を提供してくれた。

「購買部長の吉田さんは、非常に優秀な方だよ。ただ、現場上がりだから、理屈だけの提案は嫌う。データと実例がないと動かない」

「なるほど...」

「それと、社長の大野さんは『従業員第一主義』を掲げている。利益も大事だが、従業員の働きやすさを何より重視する経営者だ」

山田はメモを取りながら聞いた。

「あと、最近、品質問題があってね。大手メーカーから厳しい指摘を受けたらしい。それで、品質管理体制の見直しを進めているみたいだ」

「品質問題...」

これは重要な情報だ。

「ありがとうございます、社長。大変参考になりました」

## 4. 6つの価値による分析

その夜、山田は宮本の自宅で、集めた情報を整理していた。

宮本が見守る中、山田はホワイトボードに書き始めた。

「まず、1つ目の『機能的価値』です」

```
【1. 機能的価値】
日東自動車部品が求める機能は？

・高精度な測定機器
・リアルタイムな不良品検出
・生産ラインへの組み込み可能性
```

「良いスタートだ」宮本が頷いた。「次は？」

「2つ目、『経済的価値』」

```
【2. 経済的価値】
コスト削減・利益向上のポイントは？

・品質問題による損失削減
  └─ 大手メーカーからのクレーム対応コスト
  └─ 不良品による廃棄コスト
  └─ 検査工程の人件費

・売上向上の可能性
  └─ 品質向上による取引拡大
  └─ 新規顧客開拓の可能性
```

「具体的な数字は？」

「IR資料によると、昨年度の品質クレーム対応費用は約5,000万円。検査部門の人件費は年間1億2,000万円です」

「よく調べたな。次は？」

山田は少し考えてから書いた。

「3つ目、『感情的価値』」

```
【3. 感情的価値】
経営者・従業員が感じる感情は？

社長（大野氏）の想い：
・従業員を守りたい
・品質問題の再発を防ぎたい
・安心して眠れる体制を作りたい

購買部長（吉田氏）の想い：
・現場を疲弊させたくない
・実績のある確実な方法を選びたい

検査部門の従業員：
・ミスへの不安
・長時間労働への疲労
・もっと価値のある仕事がしたい
```

宮本の目が輝いた。

「素晴らしい。中村社長から聞いた情報を、見事に『感情』として捉えている」

「ありがとうございます」

山田は続けた。

「4つ目、『社会的価値』」

```
【4. 社会的価値】
社会・業界への貢献は？

・自動車産業全体の品質向上
・安全な自動車の提供
・製造業のDX推進事例
・地域雇用の維持・拡大
```

「日東自動車部品のホームページには、『社会に安全を提供する』という理念が書かれていました。品質向上は、その理念の実現に直結します」

「良い視点だ。では、5つ目は？」

「『体験的価値』」

```
【5. 体験的価値】
導入・使用時の体験は？

導入時：
・スムーズな導入プロセス
・現場への負担が少ない
・段階的な移行が可能

使用時：
・直感的な操作性
・トラブル時の即座な対応
・継続的なサポート

長期的：
・成長を実感できるデータ
・改善の可視化
・成功事例の共有
```

「なぜ、これが重要だと思う？」

「大企業ほど、変化を嫌います。新しいシステムの導入は、現場に大きな負担をかける。だから、『体験』が悪ければ、どんなに良い商品でも受け入れられません」

「完璧だ」

宮本は満足そうに頷いた。

「では、最後の6つ目は？」

山田は少し迷ってから書いた。

「『象徴的価値』」

```
【6. 象徴的価値】
ブランド・ステータスの意味は？

・業界トップクラスの品質管理体制を持つ企業
・先進的な技術を導入する革新的な企業
・従業員を大切にする優良企業

→ これらのイメージが、取引先や求職者から
  選ばれる理由になる
```

「素晴らしい」

宮本は立ち上がった。

「山田君、君は今、日東自動車部品を6つの視点で完全に分析した。これが『相手の立場で考える』ということだ」

## 5. 仮説の構築

宮本はコーヒーを淹れ直しながら、続けた。

「6つの価値分析ができたら、次は『仮説』を立てる」

「仮説...ですか」

「そうだ。『もし、日東自動車部品がこの6つの価値をすべて手に入れたら、どうなるか』という未来の姿を描く」

山田は新しいページを開いた。

「では、描いてみます」

```
【日東自動車部品の1年後の姿】

■ 機能的価値の実現
・最新の検査システムが稼働
・不良品検出率99.9%以上

■ 経済的価値の実現
・品質クレーム対応費用：5,000万円 → 500万円
・検査部門人件費：1億2,000万円 → 7,000万円
・年間コスト削減：約1億円

■ 感情的価値の実現
・大野社長：「これで安心して眠れる」
・吉田部長：「現場の表情が明るくなった」
・検査員：「やりがいのある仕事に集中できる」

■ 社会的価値の実現
・業界誌に成功事例として掲載
・他社からの視察依頼が増加
・地域の製造業DXのモデル企業に

■ 体験的価値の実現
・導入3ヶ月で現場が完全に習熟
・トラブル発生時も即座に対応
・毎月の改善レポートで成長を実感

■ 象徴的価値の実現
・「品質の日東」として業界での地位向上
・求人応募者数が前年比150%増加
・大手メーカーからの信頼度向上
```

宮本は黙って読んでいたが、やがてゆっくりと頷いた。

「山田君、これは完璧だ」

「本当ですか？」

「ああ。君は今、『相手の経営者になりきって』未来を描いた。これが、大企業を動かす提案の核心だ」

## 6. 提案書の構成

翌日、山田は提案書の作成に取りかかった。

宮本からのアドバイスを元に、従来とはまったく違う構成で作り始めた。

```
【日東自動車部品様 ご提案資料】

1. エグゼクティブサマリー
   「あなたの会社は、1年後にこうなります」
   → 6つの価値の実現イメージを1ページで

2. 現状分析
   「私たちが理解した、御社の現状と課題」
   → 6つの価値の視点で分析

3. 課題の本質
   「本当の課題は何か」
   → 品質問題の裏にある経営課題

4. 解決策
   「6つの価値を実現する方法」
   → 具体的なソリューション

5. 実現プロセス
   「どうやって実現するか」
   → 段階的な導入計画

6. 投資対効果
   「経済的価値の試算」
   → 1億円のコスト削減根拠

7. 成功事例
   「ナカムラ工業様の事例」
   → 実績とお客様の声

8. 私たちの約束
   「6つの価値の実現を保証します」
   → 成果保証とサポート体制
```

山田は、各ページを丁寧に作り込んでいった。

特に力を入れたのは、「エグゼクティブサマリー」だ。

経営者は忙しい。最初の1ページで心を掴まなければ、残りは読んでもらえない。

山田は、大野社長の視点に立って、最も響く言葉を選んだ。

```
【1年後、あなたの会社はこう変わります】

・品質クレームゼロの安心感
・年間1億円のコスト削減
・従業員の笑顔
・業界トップクラスの品質管理体制
・取引先からの信頼向上
・次世代を担う人材の確保

これらすべてを、私たちが実現します。
なぜなら、私たちは「商品」を売るのではなく、
「価値」を創造する会社だからです。
```

## 7. リハーサル

提案の前日、山田は林部長と宮本を相手に、プレゼンのリハーサルを行った。

「では、始めます」

山田は、深呼吸してから口を開いた。

「大野社長、本日は貴重なお時間をいただき、ありがとうございます」

「私たちは、御社の課題を6つの視点で分析しました。機能的価値、経済的価値、感情的価値、社会的価値、体験的価値、そして象徴的価値です」

林部長が手を上げた。

「ストップ。山田君、いきなり『6つの視点』は専門用語すぎる」

「あ...そうですね」

「もっとシンプルに。『御社が本当に求めているものは何か、徹底的に考えました』くらいでいい」

「分かりました」

山田はメモを修正した。

プレゼンを再開すると、今度は宮本が質問した。

「山田君、なぜあなたの会社を選ぶべきなのか、30秒で答えてくれ」

山田は一瞬戸惑ったが、すぐに答えた。

「私たちは、御社の成功に本気でコミットするからです。商品を売って終わりではなく、1年後に約束した価値をすべて実現するまで、伴走します」

宮本が微笑んだ。

「良い。その言葉、必ず入れなさい」

リハーサルは3時間に及んだ。

想定される質問、懸念点、競合との差別化。あらゆる角度から準備を重ねた。

リハーサルが終わると、宮本が山田の肩に手を置いた。

「山田君、君は十分に準備した。あとは自信を持って臨むだけだ」

「ありがとうございます」

「一つだけ、忘れないでほしい」

「何でしょうか」

「相手の立場で考える、ということは、相手になりきることだ。大野社長の不安、吉田部長の懸念、現場の疲労。すべてを自分のことのように感じながら話すんだ」

「はい」

「それができれば、必ず伝わる」

## 8. 提案当日

水曜日の午後2時。

山田は日東自動車部品の本社ビル7階の会議室にいた。

窓の外には、広大な工場が見える。フォークリフトが行き交い、従業員たちが働いている。

「お待たせしました」

ドアが開き、3人の男性が入ってきた。

中央の60代の男性が、社長の大野だろう。右側の50代が購買部長の吉田、左側の40代が品質管理部長と思われる。

「初めまして。日東自動車部品の大野です」

「東洋精機の山田と申します。本日はお時間をいただき、ありがとうございます」

全員が着席した。

山田は、資料を配る前に、一つ質問した。

「提案に入る前に、一つお聞きしてもよろしいでしょうか」

「どうぞ」

「大野社長は、この会社をどんな会社にしたいとお考えですか？」

大野社長は少し驚いた表情を見せたが、すぐに答えた。

「従業員が誇りを持って働ける会社です。品質で業界をリードし、取引先から信頼され、働く人が幸せを感じられる。そんな会社にしたい」

山田は深く頷いた。

「分かりました。今日の提案は、まさにその実現のためのものです」

山田は資料を開いた。

「私たちは、御社を6つの視点で分析しました」

スクリーンに、6つの価値の図が映し出された。

「機能、経済、感情、社会、体験、象徴。この6つすべてを実現することで、御社は大野社長の描く理想の会社になれると確信しています」

吉田部長が身を乗り出した。

山田は、一つ一つ丁寧に説明していった。

データ、実例、具体的な数字。すべてを盛り込みながら、常に「御社にとっての意味」を語った。

30分のプレゼンが終わると、会議室には静寂が訪れた。

大野社長が口を開いた。

「山田さん、質問があります」

「はい」

「なぜ、ここまで詳しく調べたのですか？まだ取引も始まっていないのに」

山田は真っ直ぐに社長の目を見た。

「私たちは、お客様の成功を本気で考えるからです。商品を売ることが目的ではありません。御社が理想の会社になること、それが私たちの目的です」

大野社長の目が潤んだ。

「...ありがとう」

社長は立ち上がり、山田の手を握った。

「正直、何社か提案を受けました。でも、ここまで私たちのことを考えてくれた会社は、あなたが初めてだ」

「ありがとうございます」

「詳細は吉田に詰めてもらいますが、基本的に御社にお願いしたいと思います」

山田は深々と頭を下げた。

## 9. 振り返り

その夜、山田は宮本に電話で報告した。

「先生、成功しました」

「おめでとう。どうだった？」

「6つの価値のフレームワーク、本当に効果がありました。相手の経営課題を多角的に理解し、提案できたことが、決め手になったと思います」

「それは良かった」

宮本の声には、満足感が滲んでいた。

「山田君、覚えておいてほしい。今日成功したのは、君が『相手の立場で考えた』からだ」

「はい」

「6つの価値というフレームワークは、単なるツールではない。それは、顧客を深く理解するための『思考の型』なんだ」

山田はメモを取りながら聞いた。

「機能だけでなく、経済も。経済だけでなく、感情も。感情だけでなく、社会も。すべての側面から顧客を見る。そうすることで、初めて『完全な理解』に到達できる」

「分かりました」

「これからも、この6つの視点を忘れるな。それが、君を一流の営業マンにする」

「はい。ありがとうございます」

電話を切った後、山田はノートに書き留めた。

```
【第10章の学び：相手の立場で考える】

■ 6つの価値フレームワーク
1. 機能的価値：商品の基本機能
2. 経済的価値：コスト削減・利益向上
3. 感情的価値：安心感・誇り・喜び
4. 社会的価値：社会貢献・CSR
5. 体験的価値：使用体験・顧客体験
6. 象徴的価値：ブランド・ステータス

■ 大企業攻略の鍵
・6つすべての視点で分析する
・経営者の視点に立って未来を描く
・相手の課題を自分のこととして感じる

■ 今日の成功要因
・徹底的な事前調査
・6つの価値による多角的分析
・相手の理想に共感した提案
```

山田は窓の外を見た。

夜空には星が輝いていた。

（これが、プロの営業か...）

まだ道は長い。だが、確実に成長している実感があった。

価格競争から価値創造へ。

その道を、山田は一歩一歩進んでいる。

## 本章の核心メッセージ

**6つの価値フレームワークで顧客を完全に理解する**

大企業を攻略するには、単一の視点では不十分だ。機能、経済、感情、社会、体験、象徴—この6つすべての側面から顧客を理解し、包括的な提案をすることが成功の鍵となる。

「相手の立場で考える」とは、相手の経営課題を自分のこととして感じ、6つの価値すべてを実現する未来を描くことである。

次章では、この6つの価値をどのように定量化し、提案書に落とし込むかの技術を学ぶ。
