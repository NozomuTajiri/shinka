## 1. 最後の訪問

山田浩一は、いつもの喫茶店に向かっていた。

一年半の旅を経て、彼は営業本部長として、そして価値創造推進室長として、会社と業界の変革を牽引する立場になっていた。業績は好調、社員は生き生きとし、取引先からの信頼も厚い。

だが、今日は特別な日だった。

宮本顧問から、「最後に話したいことがある」と連絡があったのだ。

午後三時。いつもの席に、宮本が座っていた。

「山田君、来てくれたね」

「先生、『最後に』とは、どういう……」

宮本は穏やかに微笑んだ。

「心配しないでくれ。私が死ぬわけじゃない」

「よかった……」

「ただ、君に教えるべきことは、もうほとんど教え終えた。今日は、すべての総まとめだ」

山田は、深く頷いた。

「お願いします」

宮本はいつものように、ノートを開いた。

## 2. 価値とは何か、最終回答

「山田君、最初の日を覚えているか」

「はい。先生は僕に『あなたの商品を買うことで、お客様は何を手に入れるのか』と聞きました」

「そして、君は答えられなかった」

「恥ずかしながら……」

宮本は、ゆっくりとコーヒーを一口飲んだ。

「あれから一年半。君はたくさんのことを学んだ。六つの価値、顕在ニーズと潜在ニーズ、価値の可視化、組織変革、そして価値の連鎖」

「はい」

「では、今なら答えられるか。『価値とは、何か』」

山田は、しばらく考えた。

これまで学んできたすべてが、頭の中を駆け巡った。

そして、山田は口を開いた。

「価値とは……お客様の課題を解決し、喜びを生み出すことです」

「それは正しい。でも、まだ不完全だ」

「……」

宮本は、ノートに大きく一つの言葉を書いた。

**時間**

「価値とは、究極的には『時間』なんだよ、山田君」

## 3. 時間=命という気づき

「時間……ですか」

山田は、その言葉の意味を咀嚼しようとした。

「そうだ。考えてみてくれ」

宮本は窓の外を見た。

「人間に与えられた資源は、たった一つしかない」

「資源……」

「お金でもない、能力でもない、人脈でもない」

宮本は、山田の目を見た。

「時間だ。正確に言えば、『命の時間』だ」

山田は、息を飲んだ。

「人間は、一日二十四時間しか持っていない。そして、その時間は二度と戻らない」

「……」

「つまり、時間を使うということは、命を使うということだ」

宮本の言葉が、山田の胸に深く突き刺さった。

「山田君、君が顧客を訪問する時、何を使っている?」

「時間……いえ、命を使っています」

「その通り。君の部下が残業する時、何を使っている?」

「命を……使っています」

「お客様が、君の提案を検討する時、何を使っている?」

「……命を、使っています」

宮本は、静かに頷いた。

「すべてのビジネス活動は、誰かの命の時間を使っている」

## 4. コストとしての時間、価値としての時間

宮本は、ノートに二つの言葉を書いた。

**コストとしての時間**

**価値としての時間**

「コスト……」

「『時間がかかる』『時間の無駄』『時間が惜しい』——こういう言葉を聞いたことがあるだろう」

山田は頷いた。

宮本は続けた。

山田は、深く頷いた。

「つまり、同じ二時間でも、『コスト』として見るか、『価値創造』として見るか。その視点の違いが、結果を分けるんです」

「完璧だ」

宮本の声には、深い重みがあった。

「罪だと思わないか?」

## 本章のキーメッセージ

**価値とは、究極的には『命の時間』である**

この章で、山田は価値の究極的な定義に到達した。

```
【価値と時間の関係】

価値 = 命の時間を、いかに変換するか

コストとしての時間:
└─ 長時間労働、無駄な作業、報われない努力

価値創造としての時間:
└─ 最小の時間で、最大の価値を生み出す

経営の本質:
└─ 社員の命の時間を、価値創造に使えるようにすること

人生の意味:
└─ 命の時間を輝かせること
```

**最小の命の時間で、最大の付加価値を**

これが、カクシンの究極的なミッションだった。

長時間働くことが美徳ではない。
短時間で、より大きな価値を生み出すことが美徳だ。

そのために:
- 本質に集中する
- お客様の潜在ニーズに応える
- 無駄な作業を排除する
- 社員の時間を大切にする
- お客様の時間を大切にする

すべては、「命の時間を価値創造に使う」という原則に繋がる。

**人は、価値を生み出している時、最も輝く**

山田は学んだ。

自分の命の時間が、意味のあることに使われていると感じた時、人は幸せになる。

逆に、命の時間が無駄に消費されていると感じた時、人は不幸になる。

だから、リーダーの仕事は:
- 社員の命の時間を、価値創造に使えるようにすること
- 無駄な作業を排除すること
- 本質的な仕事に集中できる環境を作ること

そうすれば、社員は輝く。
そして、その輝きが、お客様に、取引先に、社会に広がっていく。

**究極の問い**

```
あなたの命の時間は、何のために使われているか?
```

この問いに答えられた時、人は初めて自分の人生を生きることができる。

山田は、この問いに答えることができるようになった。

「自分の命の時間は、価値を創造するために使われている」

だから、彼は幸せだった。
だから、彼の人生は輝いていた。

あなたの命の時間は、今、何のために使われているだろうか?

それは、本当に価値を生み出しているだろうか?

もし答えが「ノー」なら、変える時だ。

最小の命の時間で、最大の付加価値を。

それが、あなたの人生を輝かせる。
