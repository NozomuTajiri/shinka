# 第21章 宮本の告白

価値を見失った者だけが、価値の本質に辿り着ける

---

## 1. 突然の招待

「山田君、今晩、時間はあるか？」

講演会場を出た山田に、宮本が静かに声をかけた。

いつもの穏やかな口調だったが、どこか違う雰囲気があった。

「はい、大丈夫です」

「では、一緒に食事でもどうだ。話したいことがある」

山田は少し驚いた。これまで一年半、宮本とは数え切れないほど会ってきた。だが、宮本のほうから食事に誘われたのは、これが初めてだった。

午後七時。二人は、都内の静かな日本料理店の個室にいた。

窓の外には、東京タワーが淡く光っている。

「乾杯」

宮本がグラスを掲げた。

「乾杯」

山田も応じた。

料理が運ばれてくる。季節の先付け、お椀、お造り。丁寧に作られた一品一品が、目も舌も楽しませてくれる。

だが、宮本はいつもより静かだった。

## 2. 語られなかった過去

「山田君」

宮本が箸を置いた。

「私には、君に話していないことがある」

「......」

山田は、宮本の表情を見つめた。

七十二歳。穏やかで、知性に溢れ、常に的確な助言をくれる恩師。だが、今夜の表情には、どこか影があった。

「実は私も、かつて価値を見失った人間だったんだ」

「え......？」

山田は驚いた。

宮本が？ あの、価値創造の理論を教えてくれた宮本が？

「信じられないかもしれないが、本当だ」

宮本は遠い目をした。

「それどころか、私は人生最大の失敗をした。顧客を裏切り、会社を追われ、すべてを失った」

山田は言葉を失った。

## 3. 成功に溺れた日々

「四十年前の話だ」

宮本は、ゆっくりと語り始めた。

「私は当時、キーエンスという会社にいた。創業間もない頃だ」

「キーエンス......」

山田は息を飲んだ。日本を代表する計測機器メーカー。高収益企業として知られ、営業力の強さは業界の伝説だ。

「私は、営業マンとして入社した。三十歳のときだ」

宮本は続けた。

「当時のキーエンスは、まだ小さな会社だった。だが、創業者の滝崎武光さんは、明確なビジョンを持っていた」

「どんなビジョンですか？」

「『お客様の課題を解決する』——ただそれだけだ。製品を売るのではなく、価値を届ける。そのために、徹底的に顧客の現場を理解する」

山田は深く頷いた。まさに、自分が宮本から学んだことだ。

「私は、その理念に共感した。そして、必死に働いた」

宮本の目が、遠い過去を見つめている。

「朝から晩まで顧客を回り、課題を聞き、解決策を提案する。夜は社内で勉強会。週末も工場見学。寝る時間もないくらい働いた」

「......」

「でも、苦しくはなかった。なぜなら、お客様が本当に喜んでくれたからだ」

宮本は、少し笑った。

「『宮本さんのおかげで助かったよ』『この機器で生産性が三割上がった』『従業員が早く帰れるようになった』——そういう声をもらうたびに、心が満たされた」

山田は、自分の体験と重ね合わせていた。

「そして、私は成果を出した。入社三年目で、営業部のトップになった。五年目には、課長に昇進した。十年目には、部長になった」

「素晴らしいじゃないですか」

「ああ、順風満帆だった」

宮本は、苦い表情を浮かべた。

「そして、そこで私は変わってしまったんだ」

## 4. 傲慢の始まり

「部長になって、私は大きな権限を持つようになった」

宮本はグラスの水を一口飲んだ。

「営業戦略を決める。人事を動かす。大口顧客との交渉を仕切る。会社の中で、重要な存在になった」

「それは当然ですよね。実績があったんですから」

「いや」

宮本は首を振った。

「そこが罠だったんだ」

「罠......？」

「成功すると、人間は勘違いする。『自分が優秀だから成功した』と」

宮本は、自嘲するように笑った。

「だが、違う。成功の本質は、『お客様に価値を届けたから』だ。自分の能力ではなく、お客様の喜びが原点だった」

「......」

「でも、私はそれを忘れた。いつの間にか、『自分が優秀だから売れるんだ』と思い込むようになっていた」

山田は、胸が締め付けられるような気持ちになった。

「そして、私は傲慢になった」

宮本の声が、わずかに震えた。

「お客様の話を聞かなくなった。『この製品が最適です』と一方的に提案するようになった。反論されると、『あなたは分かっていない』と見下した」

「宮本さんが......そんなことを」

「信じられないだろう。でも、本当なんだ」

## 5. 裏切り

「そして、ある日——」

宮本は深く息を吸い込んだ。

「大口顧客から、クレームが入った」

「クレーム......」

「私が提案した測定機器が、まったく役に立たなかったんだ」

宮本の拳が、わずかに震えた。

「その顧客は、中堅の自動車部品メーカーだった。社長の木村さんは、私を信頼してくれていた。『宮本さんが勧めるなら間違いない』と言って、三千万円の投資を決断してくれた」

「......」

「でも、私は木村社長の本当の課題を聞いていなかった。『この機器が最新だから』『業界で評価されているから』——そんな理由だけで、押し売りしたんだ」

山田は、息を飲んだ。

「導入して三ヶ月後、木村社長から電話があった」

宮本は目を閉じた。

「『宮本さん、この機器、うちには合わなかった』——そう言われたとき、私は何と答えたと思う？」

「......」

「『使い方が悪いんじゃないですか』と言ったんだ」

山田は言葉を失った。

「木村社長は、それ以上何も言わなかった。ただ、『分かりました』とだけ言って、電話を切った」

宮本の目が、わずかに潤んでいた。

「その一週間後、木村社長の会社は倒産した」

「......！」

「三千万円の投資が、資金繰りを圧迫していたんだ。もし、私が本当に木村社長の課題を聞いていたら——もし、本当に必要な提案をしていたら——」

宮本の声が、震えた。

「木村社長の会社は、救えたかもしれない」

## 6. 失墜

「その事実を知ったとき、私は崩れ落ちた」

宮本は、拳を握りしめた。

「私は、顧客を裏切った。木村社長を裏切った。従業員五十人の生活を奪った」

「......」

「会社にも、報告せざるを得なかった。上司から、厳しく叱責された。滝崎社長にも呼ばれた」

宮本は、遠い目をした。

「社長は、静かに言った。『宮本君、君は何のために営業をしているのか』」

「......」

「私は答えられなかった。そして、社長はこう続けた」

宮本は、その言葉を一語一語、噛みしめるように話した。

「『君は、製品を売ることが目的になっていた。でも、私たちの仕事は、お客様の課題を解決することだ。それを忘れた者に、営業を任せることはできない』」

山田は、息を飲んだ。

「私は、役職を剥奪された。部長から、平社員に降格された」

「......」

「そして、半年後——私は会社を辞めた」

宮本は、静かに言った。

「いや、正確には、辞めざるを得なかった。周囲の目が冷たくなり、居場所がなくなったんだ」

## 7. どん底

「四十二歳で無職になった」

宮本は、苦笑した。

「妻と幼い娘がいた。住宅ローンも残っていた。でも、仕事がなかった」

「......」

「再就職を試みたが、うまくいかなかった。面接で『前職を辞めた理由』を聞かれるたびに、答えに窮した」

宮本は続けた。

「貯金は、半年で底をついた。家を売った。娘の進学も諦めさせた」

「宮本さん......」

「一番辛かったのは」

宮本の声が、かすれた。

「妻の目だった。最初は励ましてくれた。『大丈夫よ、あなたなら立ち直れる』と。でも、三年経っても状況が変わらないと、彼女の目から光が消えていった」

山田は、何も言えなかった。

「そして、ある日——」

宮本は、グラスを握りしめた。

「娘が、中学校でいじめられた。『お父さん、無職なんだって』と」

「......」

「娘は泣きながら帰ってきた。私は、何もしてやれなかった」

宮本の目から、一筋の涙が流れた。

「その夜、私は本気で死のうと思った」

山田は、息を飲んだ。

「でも——」

宮本は、涙を拭った。

「娘が、私の部屋に来たんだ。『パパ、大丈夫。私、パパのこと信じてるから』と」

「......」

「その言葉に、私は救われた」

## 8. 再生への道

「翌日、私は決意した」

宮本の目に、光が戻った。

「もう一度、ゼロから学び直す。そして、本当の価値とは何かを理解する」

「どうされたんですか？」

「図書館に通い詰めた。マーケティング、経営学、心理学——あらゆる本を読んだ」

宮本は続けた。

「そして、ピーター・ドラッカーの言葉に出会った」

「ドラッカー......」

「『企業の目的は、顧客を創造することである』『顧客が価値と認めるものを提供すること、それが企業の使命である』」

宮本は、目を閉じた。

「その言葉を読んだとき、私は泣いた」

「......」

「私は、何も分かっていなかった。顧客を創造するどころか、顧客を壊していた。価値を提供するどころか、価値を奪っていた」

宮本は、深く息を吐いた。

「そして、私は誓った。もう二度と、顧客を裏切らない。本当の価値を届ける人間になる」

## 9. 学びの日々

「それから、私は徹底的に学んだ」

宮本は語った。

「価値とは何か。顧客の本当のニーズとは何か。どうすれば、本質を見抜けるのか」

「......」

「そして、私は気づいたんだ」

宮本は、山田の目を見た。

「価値とは、六つの側面がある。機能、経済、感情、社会、体験、象徴——これらすべてを理解して初めて、顧客の本当のニーズに応えられる」

山田は、深く頷いた。自分が学んだ「六つの価値」の理論だ。

「この理論は、私が三年かけて構築したものだ」

「三年......」

「ああ。木村社長の失敗を二度と繰り返さないために、必死に考え抜いた」

宮本は、優しく微笑んだ。

「そして、四十五歳のとき、私は小さなコンサルタント会社を立ち上げた」

「それが......」

「ああ。『価値創造支援』を専門にする会社だ」

## 10. なぜ山田を指導したのか

「山田君」

宮本は、真剣な表情になった。

「なぜ、私が君を指導してきたのか、分かるか？」

「......」

山田は、黙って首を振った。

「君の中に、かつての私を見たからだ」

「え......？」

「一年半前、君が研修に来たとき——覚えているか？」

「はい」

「君は疲れ切っていた。努力しても報われない。何のために働いているのか分からない。そういう絶望が、顔に出ていた」

山田は、あの日の自分を思い出した。

「でも、同時に——君の目には、まだ光があった」

宮本は続けた。

「『本当は、顧客の役に立ちたい』『価値のある仕事がしたい』——そういう純粋な願いが、君の奥底に残っていた」

「......」

「私は思ったんだ。この若者を、かつての私のように失敗させてはいけない、と」

宮本の目が、潤んだ。

「私は、木村社長を救えなかった。でも、君なら——君なら、多くの顧客を救える」

「宮本さん......」

「だから、私は君に、すべてを教えた」

宮本は、山田の肩に手を置いた。

「六つの価値の理論。本質を掘り下げる質問法。価値を見える化する方法。そのすべてを」

山田の目から、涙が溢れた。

「先生......」

「山田君、君は私の誇りだ」

宮本は、優しく微笑んだ。

「君は、この一年半で、何十社もの顧客に価値を届けた。何百人もの人々の働き方を変えた。そして、業界全体に影響を与え始めている」

「......」

「それが、私の贖罪なんだ」

宮本の声が、震えた。

「私は、木村社長を救えなかった。でも、君を通じて——君が救った顧客を通じて——私は少しずつ、罪を償っている」

## 11. 師弟の絆

山田は、立ち上がった。

そして、深々と頭を下げた。

「先生、ありがとうございます」

「顔を上げてくれ」

宮本も立ち上がった。

「私こそ、感謝している。君が、私の理論を実践してくれた。君が、価値の連鎖を広げてくれた」

二人は、しばらく向かい合って立っていた。

窓の外では、東京の夜景が輝いている。

「山田君」

宮本が、静かに言った。

「私が君に伝えたいことは、一つだけだ」

「はい」

「価値を見失うことは、誰にでもある」

宮本は、遠い目をした。

「成功すれば、傲慢になる。忙しくなれば、本質を見失う。疲れれば、手を抜きたくなる」

「......」

「でも、大切なのは——見失ったとき、どうするかだ」

宮本は、山田の目を真っ直ぐ見た。

「立ち止まる勇気を持て。自分に問いかける誠実さを持て。そして、もう一度、顧客の声に耳を傾ける謙虚さを持て」

「はい」

「それができれば——どんな失敗からも、立ち直れる」

宮本は、優しく微笑んだ。

「私がそうだったように」

## 12. 新たな誓い

その夜、帰りの電車の中で、山田はノートに書いた。

```
【宮本先生から学んだこと】

■ 成功は、傲慢の始まりになりうる
  └─ 常に謙虚さを忘れない

■ 価値を見失うことは、誰にでもある
  └─ 立ち止まり、問い直す勇気を持つ

■ 失敗は、再生のチャンスである
  └─ 本質を学び直す機会と捉える

■ 教えることは、贖罪であり、使命である
  └─ 学んだことを、次の世代に伝える
```

山田は、ペンを置いた。

そして、心の中で誓った。

（私は、宮本先生のようにはならない）

（いや——）

山田は、考え直した。

（私も、いつか失敗するかもしれない。傲慢になるかもしれない。価値を見失うかもしれない）

（でも、そのときは——）

（先生のように、立ち直る。そして、学び直す。そして、次の世代に伝える）

窓の外を、夜の景色が流れていく。

山田は、深く息を吸い込んだ。

（先生が私を救ってくれたように、私も誰かを救う）

（それが、価値の連鎖だ）

（それが、本当の意味での『価値創造』だ）

## 13. 翌朝の決意

翌朝、山田は早めに出社した。

そして、自分のデスクに座り、ノートを開いた。

宮本先生の告白を受けて、自分がすべきことを整理する。

```
【私の新しい使命】

1. 常に顧客起点であり続ける
   └─ 成功しても、傲慢にならない
   └─ 忙しくても、本質を見失わない

2. 価値の連鎖を広げ続ける
   └─ 社内、取引先、業界全体へ
   └─ 一人ひとりの人生を変える

3. 次世代を育てる
   └─ 学んだことを、惜しみなく伝える
   └─ 宮本先生がしてくれたように

4. 失敗から学ぶ文化を作る
   └─ 失敗を隠さない
   └─ 失敗を成長の機会とする
```

山田は、ノートを閉じた。

窓の外には、朝日が昇っている。

新しい一日が、始まる。

そして、山田浩一の新しい使命も、始まる。

## エピローグ：師弟を超えて

数日後、山田は若手社員の田中を呼んだ。

「田中君、君に任せたいプロジェクトがある」

「はい！」

田中は、目を輝かせた。

「日東自動車部品の次のフェーズだ。品質管理システムの拡張提案を、君が担当してほしい」

「え......僕が、ですか？」

田中は驚いた。

「ああ。君には素質がある。でも、まだ経験が足りない」

山田は、微笑んだ。

「だから、私が教える。一から、丁寧に」

「ありがとうございます！」

田中は、深々と頭を下げた。

山田は、その姿に、一年半前の自分を見た。

（これが、価値の連鎖だ）

（宮本先生から私へ。私から田中へ。田中から、次の誰かへ）

（こうして、価値は広がっていく）

山田は、窓の外を見た。

青空が、どこまでも広がっている。

そして、その空の向こうに——

宮本先生の笑顔が、見えるような気がした。

## 本章のキーメッセージ

**価値を見失った者だけが、価値の本質に辿り着ける**

宮本顧問は、かつて大きな失敗を経験していた。傲慢になり、顧客を裏切り、人生のどん底に落ちた。

だが、その失敗こそが、彼を「価値創造の伝道師」にした。

```
【失敗からの再生プロセス】

1. 成功 → 傲慢 → 失敗
   └─ 誰にでも起こりうる罠

2. 絶望 → 内省 → 学び直し
   └─ 本質を問い直す機会

3. 理論構築 → 実践 → 伝承
   └─ 失敗を価値に変える

4. 次世代育成 → 贖罪 → 使命
   └─ 教えることで、自分も救われる
```

宮本は、山田に自分の失敗を語ることで、最後の教えを与えた。

「成功しても、決して傲慢になるな」
「失敗したら、学び直せ」
「そして、学んだことを、次の世代に伝えよ」

山田は、この教えを胸に、新たな使命を見つけた。

価値の連鎖は、世代を超えて続く。

それが、本当の意味での「価値創造」である。
