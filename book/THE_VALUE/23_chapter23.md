# 第23章 新しい朝

変わった会社、変わった家庭、変わった自分

---

## 1. 穏やかな朝

午前六時三十分。

山田浩一は、目覚まし時計が鳴る前に目を覚ました。

かつて、朝は戦いだった。重い体を無理やり起こし、疲れを引きずったまま会社へ向かう。それが、二十年以上続く日常だった。

だが、今は違う。

自然に目が覚め、清々しい気持ちで一日を始められる。

窓を開けると、初夏の爽やかな風が部屋に入ってきた。庭の木々が、朝日を浴びて輝いている。

「おはよう」

妻の恵子が、キッチンから声をかけてきた。

「おはよう」

リビングに行くと、朝食の準備ができていた。焼きたてのトースト、サラダ、コーヒーの香り。

「今日は早いのね」

「うん。朝のウォーキングに行ってくるよ」

「あら、いつから?」

「今日から」山田は笑った。「以前から思ってたんだ。朝の時間を大切にしようって」

恵子は、驚いたように夫を見た。

二年前までの山田なら、考えられなかった。朝は一分でも長く寝ていたい。ギリギリまで布団にしがみついていた。

「行ってらっしゃい」

「行ってくる」

---

## 2. 変わった街、変わった自分

近所の公園を歩きながら、山田は深呼吸をした。

空気が美味しい。鳥のさえずりが心地よい。

ベンチに座って、スマートフォンを取り出した。メールをチェックする。

```
件名: 【感謝】価値創造プログラムの成果報告

山田室長

いつもお世話になっております。
タカハシ工業の高橋です。

先日ご指導いただいた価値提案手法を実践したところ、
長年取引のなかった大手メーカーから
大型受注をいただくことができました。

担当者の方が、「こんなに自社のことを考えてくれる
提案は初めてだ」と涙ぐんでおられました。

山田室長のおかげで、私たちは「営業の本質」を
学ぶことができました。

本当にありがとうございます。
```

山田は、静かに微笑んだ。

二年前まで、こんなメールは一通も来なかった。

それどころか、クレームや値下げ要求のメールばかりだった。

でも、今は違う。

毎日のように、感謝のメールが届く。

「価値を届けるって、こういうことなんだな」

山田は、空を見上げた。

青く澄んだ空が、どこまでも広がっていた。

## 3. 会社の朝

午前八時。

大和精密工業のエントランスに着くと、警備員の田中さんが笑顔で迎えてくれた。

「おはようございます、山田室長」

「おはようございます、田中さん」

エレベーターに乗ると、若手社員が二人、楽しそうに話していた。

「昨日の商談、すごく手応えがあったよ」

「マジ? どんな提案したの?」

「六つの価値のうち、『社会的意義』と『感情的満足』を組み合わせたんだ。そしたら、社長が『まさに、それが欲しかった』って」

「おお! やるじゃん!」

二人は山田に気づくと、会釈した。

「おはようございます」

「おはよう。良い商談だったみたいだね」

「はい! すごく楽しかったです!」

若手の目が、輝いていた。

山田は、心の中で頷いた。

二年前、営業部の若手たちは疲弊していた。長時間労働、価格競争、達成できないノルマ。

でも、今は違う。

全員が、誇りを持って働いている。

「楽しい」と言いながら、営業をしている。

それが、何よりの成果だった。

オフィスに着くと、価値創造推進室のメンバーが既に出社していた。

「おはようございます」

佐藤課長、北部部長、そして新たに加わった三名の専任スタッフ。

「おはようございます。本部長」

デスクには、今日のスケジュールが並んでいた。

```
【本日の予定】

09:00 - 新規支援企業との初回ミーティング
11:00 - 価値創造プログラム第12期・成果報告会
14:00 - 業界誌の取材対応
16:00 - 宮本顧問との定例ミーティング
18:00 - 社内・価値創造文化浸透度調査レビュー
```

「今日も充実した一日になりそうだな」

山田は、コーヒーを淹れながら言った。

佐藤が資料を持ってきた。

「本部長、先月の支援企業の成果が出ました」

```
【価値創造プログラム 第11期成果】

参加企業: 42社
プログラム完遂率: 98%

平均成果:
- 受注額増加: 52%
- 利益率改善: 28%
- 社員満足度向上: 3.2 → 4.5
- 顧客満足度向上: 3.5 → 4.7

特筆すべき事例:
- 従業員8名の小規模メーカーが、大手企業との新規取引開始
- 赤字寸前だった企業が、3ヶ月で黒字転換
- 離職率50%だった企業が、ゼロに
```

「素晴らしいな」

山田は、一つひとつの数字に目を通した。

しかし、彼が最も注目したのは、数字ではなかった。

「この『社員の声』の欄を見てくれ」

佐藤が、ページをめくった。

```
【参加企業・社員の声】

「初めて、自分の仕事に誇りを持てました」
（製造業・営業担当・28歳）

「お客様から『ありがとう』と言われて、
涙が出ました」
（部品メーカー・技術者・35歳）

「毎日、会社に行くのが楽しみです」
（精密機器メーカー・管理職・42歳）
```

「これだよ」山田は言った。「これが、本当の成果だ」

## 5. 成果報告会

午前十一時。会議室には、第12期プログラムを修了した企業の代表者が集まっていた。

「皆さん、三ヶ月間お疲れ様でした」

山田が、開会の挨拶をした。

「今日は、それぞれの成果を共有し、学び合う場です。では、最初の発表をお願いします」

最初に立ったのは、従業員十五名の金属加工メーカー、ヤマグチ製作所の山口社長だった。

「当社は、創業五十年になります」

山口社長は、穏やかに語り始めた。

「しかし、ここ数年は価格競争に巻き込まれ、利益がどんどん減っていました」

「従業員の給料も上げられず、ボーナスもカット。社員のモチベーションは最低でした」

会場が、静まり返った。

「でも、このプログラムで学んだ『価値起点』の考え方を実践して——」

山口社長は、目を輝かせた。

「変わりました」

スクリーンに、成果が映し出された。

```
【ヤマグチ製作所の成果】

受注額: 前年比210%
利益率: 8% → 32%
従業員満足度: 2.8 → 4.8

具体的な取り組み:
1. 全ての既存顧客に「本当の課題」をヒアリング
2. 価格競争から脱却し、価値提案に転換
3. 技術力を活かした「リスク回避」提案で差別化
```

「特に、長年取引のあったお客様が言ってくれたんです」

山口社長は、声を震わせた。

「『山口さんのところは、ただ作るんじゃなくて、うちのことを本当に考えてくれる。だから、もう他には頼まない』と」

「それを聞いたとき......五十年、この仕事をやってきて良かったって、心から思いました」

拍手が起きた。

山田は、静かに頷いた。

これが、価値起点経営の力だ。

単に売上が増えるだけじゃない。

働く人の人生が、変わるんだ。

## 6. 宮本顧問との対話

午後四時。

山田は、宮本顧問の研究室を訪れた。

二年前、すべてが始まった場所。

「先生、お久しぶりです」

「山田君、元気そうだね」

宮本は、いつもの温かい笑顔で迎えてくれた。

「今日は、報告があります」

山田は、この半年間の成果をまとめた資料を渡した。

```
【価値創造の広がり（直近6ヶ月）】

直接支援企業: 156社
影響を受けた企業（推定）: 12,000社以上
関わった社員数（推定）: 120万人以上

業種の広がり:
- 製造業: 68%
- サービス業: 18%
- 建設業: 8%
- その他: 6%

地域の広がり:
- 全国47都道府県に展開
- 海外からの問い合わせ: 23件
```

宮本は、資料に目を通しながら、深く頷いた。

「素晴らしい......本当に、素晴らしい」

「先生のおかげです」

「いや」宮本は首を振った。「君が、実践したんだ。君が、証明したんだ」

宮本は、窓の外を見た。

「山田君、君は覚えているか? 二年前、初めてここに来たときのことを」

「はい」

山田は、鮮明に思い出した。

疲れ切って、希望を失っていた自分。

「あのとき、君は言った。『努力しても報われない』と」

「恥ずかしい限りです」

「いや」宮本は優しく言った。「あの言葉が、すべての始まりだった」

「......」

「君の悩みは、日本中の、いや世界中の働く人たちの悩みだった」

宮本は、山田の目を見た。

「そして、君は証明したんだ」

「何をでしょうか」

「『努力の方向を変えれば、人生が変わる』ということを」

宮本は続けた。

「価格ではなく価値で勝負する。数ではなく質で勝負する。そして、お客様に本当の価値を届ける」

「その結果、売上も上がる。利益も上がる。でも、それ以上に——」

宮本は、資料の「社員の声」のページを指さした。

「働く人たちが、幸せになる」

「これが、本当の成功だ」

山田は、深く頷いた。

「先生、一つお聞きしてもいいですか」

「何だい?」

「私は......この先、どこを目指せばいいのでしょうか」

山田は、正直な気持ちを吐露した。

「確かに、多くの企業を支援できました。多くの人に影響を与えられました」

「でも、まだ届いていない人たちが、たくさんいます」

「私は、もっと多くの人に、この『価値起点』の考え方を伝えたい」

宮本は、静かに微笑んだ。

「山田君、君はもう気づいているんじゃないか?」

「......?」

「次に何をすべきか」

## 7. 新しい決意

その夜、山田は自宅の書斎で、ノートパソコンを開いた。

画面には、真っ白なドキュメントが表示されている。

タイトルを入力した。

```
「価値起点経営 ── 働く人が幸せになる経営の本質」

著者: 山田浩一
監修: 宮本誠一郎
```

そうだ。

次にやるべきことは、この学びを「本」にすることだ。

より多くの人に届けるために。

日本中の、世界中の、疲弊している働く人たちに届けるために。

山田は、キーボードに指を置いた。

```
はじめに

私は、かつて疲弊していた。

毎日夜遅くまで働き、土日も返上し、
必死に努力した。

しかし、成果は出なかった。

「努力しても報われない」

そう思っていた。

でも、それは間違いだった。

努力の方向が、間違っていたのだ。

この本は、私が学び、実践し、証明した
「価値起点経営」のすべてを記したものである。

読者の皆さんが、私と同じように
「努力の方向」を変え、
人生を変えるきっかけになれば——

それが、私の願いである。
```

山田は、文章を書き続けた。

宮本顧問から学んだこと。
自分が実践したこと。
チームが変わったこと。
会社が変わったこと。
業界が変わったこと。

そして、何より——

一人ひとりの人生が、どう変わったか。

気づけば、午前二時を過ぎていた。

「もうこんな時間か」

でも、疲れは感じなかった。

むしろ、心が満たされていた。

これが、本当にやりたいことをしている時の感覚なんだ。

## 8. 家族との幸せ

翌朝、リビングで朝食を取っていると、娘の結衣が話しかけてきた。

「パパ、昨日遅くまで起きてたでしょ」

「ああ......バレたか」

「何してたの?」

「本を書いてたんだ」

「本? パパが?」

山田は頷いた。

「これまで学んだことを、本にして、もっと多くの人に届けたいと思ってね」

恵子が、コーヒーを注ぎながら言った。

「あなた、本当に変わったわね」

「そうかな」

「二年前は、毎日疲れた顔で帰ってきて、週末も寝てばかりだったわ」

結衣も頷いた。

「うん。パパ、いつもイライラしてたし」

「ごめんな......」

「ううん」結衣は首を振った。「今のパパの方が、ずっといい」

「目がキラキラしてるもん」

山田は、家族の顔を見た。

妻も、娘も、穏やかに笑っている。

ああ、これが本当の幸せなんだな。

高い給料でも、立派な肩書きでもない。

自分が誇れる仕事をして、家族と穏やかな時間を過ごせること。

それが、本当の幸せなんだ。

「今日、早く帰れる?」

恵子が聞いた。

「うん。六時には帰るよ」

「じゃあ、三人で外食しましょう」

「いいね!」

結衣が笑顔で言った。

「これは......」

記者が質問した。

「リーディングカンパニー......ですか」

山田は、少し考えた。

「確かに、売上や利益は伸びました。でも、私たちが本当に目指しているのは——」

「はい」

「『価値創造のリーディングカンパニー』です」

山田は、言葉を選びながら続けた。

「規模や売上で業界一番になることが、目標ではありません」

「そうではなく、『お客様に最も価値を届けられる企業』『社員が最も幸せに働ける企業』『業界全体を良い方向に導ける企業』——」

「それが、私たちの目指すリーディングカンパニーです」

「素晴らしい理念ですね」

「理念だけじゃありません。実践しています」

山田は、オフィスを案内した。

生き生きと働く社員たち。
明るい笑い声。
活発な議論。

「見てください。この風景が、すべてを物語っています」

「確かに......雰囲気が全然違いますね」

## 11. 内面的成長

その夜、山田は一人、オフィスに残って窓の外を見ていた。

夕暮れの空が、オレンジ色に染まっている。

二年前の自分を思い出す。

疲弊して、希望を失って、何のために働いているのかわからなかった自分。

でも、今は違う。

宮本顧問から学んだ「価値の原理」。

それを実践して、人生が変わった。

いや、正確に言えば——

「自分が変わったんだな」

山田は、静かに呟いた。

外側が変わったのではない。

内側が変わったんだ。

```
【山田浩一の変化】

Before:
- 成果のために働く
- 評価されるために努力する
- 報われないと不満を持つ
- 疲弊して、家族にも優しくできない

After:
- 価値を届けるために働く
- お客様の喜びが報酬
- 仕事に誇りを持つ
- 充実して、家族も大切にできる
```

「パラダイムシフト」

宮本顧問が言っていた言葉を思い出す。

物の見方が変わると、世界が変わる。

価格競争の世界から、価値創造の世界へ。

数の世界から、質の世界へ。

そして——

義務の世界から、使命の世界へ。

「俺の使命は......」

山田は、夕焼けに向かって呟いた。

「価値を届けること」

「一人でも多くの人に、価値起点の考え方を伝えること」

「そして、一人でも多くの人が、誇りを持って働けるようにすること」

それが、山田浩一の新しい使命だった。

## 12. 新しい挑戦への決意

翌週、山田は全社員を集めた総会で壇上に立った。

「皆さん、今日は重要な発表があります」

五百名を超える全社員が、期待の眼差しで山田を見つめている。

「私たちは、この二年間で大きく変わりました」

スクリーンに、二年前と現在の比較が映し出された。

```
【大和精密工業の変化】

2年前:
- 売上減少、利益圧迫
- 長時間労働、疲弊した組織
- 離職率15%
- 価格競争に巻き込まれる

現在:
- 売上・利益ともに大幅増
- 働きやすい環境、充実した社員
- 離職率2%
- 価値提案で顧客から選ばれる
```

「この変化は、皆さん一人ひとりが『価値起点』を実践してくれたからです」

拍手が起きた。

「しかし——」

山田は、声のトーンを変えた。

「これで満足してはいけません」

会場が、静まり返った。

「なぜなら、私たちが届けるべき価値は、まだまだあるからです」

「私たちの取引先に」
「私たちの業界に」
「そして、日本中の、世界中の企業に」

山田は、新しいスライドを映した。

```
【大和精密工業 新ビジョン】

ミッション:
「価値創造を通じて、働く人々の幸せを実現する」

2030年までの目標:
1. 10,000社の企業に価値創造を支援
2. 100万人の働く人々に影響を与える
3. 価値創造経営を「当たり前」の標準にする
```

どよめきが起きた。

「大きな目標です。でも、できます」

山田は、力を込めて言った。

「なぜなら、私たちは既に証明したからです」

「価値起点で考えれば、個人が変わる」
「個人が変われば、組織が変わる」
「組織が変われば、業界が変わる」
「業界が変われば、社会が変わる」

「私たちは、その第一歩を踏み出しました」

「そして今、次の大きな一歩を踏み出すときです」

山田は、全社員を見渡した。

「皆さん、一緒に歩いていただけますか?」

会場全体が、総立ちで拍手した。

それは、決意の拍手だった。

---

## 13. 希望に満ちた結末

その日の夕方、山田は宮本顧問と、久しぶりに近くのカフェで会った。

「先生、新しいビジョンを発表しました」

「聞いたよ。十年で一万社、百万人——」

宮本は、温かく微笑んだ。

「山田君らしい、大きな目標だ」

「できると思いますか?」

「君ならできる」

宮本は、即答した。

「いや、正確に言えば——」

「はい」

「君一人じゃない。君が育てた仲間たちと一緒なら、必ずできる」

山田は、深く頷いた。

「先生、一つお礼を言わせてください」

「何だい?」

「二年前、私は迷子でした」

山田は、正直な気持ちを語った。

「何のために働いているのか、わからなかった」

「努力しても報われない。そう思っていました」

「でも、先生が教えてくれました」

「方向を変えれば、人生が変わるって」

山田は、宮本の目を見た。

「先生のおかげで、私の人生は変わりました」

「本当に、ありがとうございました」

宮本は、優しく微笑んだ。

「山田君、礼を言うのは私の方だよ」

「え......?」

「私は、理論を教えた。でも、それを実践し、証明したのは君だ」

宮本は続けた。

「そして、君は今、その学びを数万人、数十万人に届けようとしている」

「これ以上の喜びはない」

二人は、静かにコーヒーを飲んだ。

窓の外には、夕日が沈んでいく。

そして、新しい夜が始まろうとしていた。

いや、夜ではない。

これは、新しい朝の前触れなんだ。

---

## 14. エピローグ ── 新しい朝

それから三ヶ月後。

山田浩一の著書『価値起点経営』が出版された。

初版一万部は、発売一週間で完売。
すぐに増刷が決まった。

全国の書店で平積みにされ、
ビジネス書ランキング一位を獲得した。

そして、何より——

読者からの反響が、凄まじかった。

```
「この本に救われました」
「働く意味が、やっとわかりました」
「明日から、価値起点で考えます」

全国から、何千通ものメッセージが届いた。
```

山田のもとには、講演依頼が殺到した。

企業、大学、自治体——

様々な場所から、声がかかった。

ある日、山田は地方の小さな町で講演をした。

聴衆は五十名ほど。

小さな製造業、商店街の店主、農家の方々。

「皆さん、私はかつて疲弊していました」

山田は、いつものように正直に語った。

「でも、『価値起点』を知って、人生が変わりました」

「そして、今日、皆さんにお伝えしたいんです」

山田は、聴衆を見渡した。

「どんなに小さな仕事でも、価値を届けられます」

「どんなに小さな会社でも、お客様を幸せにできます」

「大切なのは、規模じゃない。心です」

講演が終わると、一人の老人が近づいてきた。

「先生、ありがとうございました」

七十代くらいの、小さな工務店の社長だった。

「私は、もう引退を考えていたんです」

「後継者もいないし、売上も減ってきて......」

老人は、涙ぐんだ。

「でも、今日の話を聞いて、気づきました」

「私は、価格で勝負していた。でも、本当は——」

「お客様の家を、幸せな場所にすること。それが、私の仕事だったんです」

老人は、深く頭を下げた。

「もう一度、頑張ってみます」

山田は、老人の手を握った。

「応援しています」

## 15. 朝日の中で

ある日の早朝。

山田は、いつものように公園を歩いていた。

東の空が、朝日で赤く染まり始めている。

スマートフォンが鳴った。

見ると、見知らぬ番号からのメッセージだった。

```
山田様

初めまして。
神奈川県で小さな印刷会社を経営している者です。

先日、『価値起点経営』を読ませていただきました。

正直に言います。
読みながら、泣きました。

私も、山田様と同じように、
疲弊していました。

毎日、価格競争に巻き込まれ、
利益は減り、社員にも給料を上げられず、
何のために働いているのかわからなくなっていました。

でも、本を読んで、気づきました。

「方向が間違っていたんだ」と。

今日から、価値起点で考えます。

お客様に、本当の価値を届けます。

そして、いつか山田様のように、
この学びを他の人にも伝えられるように
なりたいです。

本当に、ありがとうございました。
```

山田は、画面を見つめた。

そして、静かに微笑んだ。

「伝わっている......」

価値は、連鎖している。

山田から、仲間へ。
仲間から、取引先へ。
取引先から、業界へ。
業界から、社会へ。

そして今——

本を通じて、見知らぬ人々へ。

空を見上げると、太陽が昇り始めていた。

新しい朝だ。

山田浩一にとっても、
日本中の働く人たちにとっても、
新しい朝が始まろうとしていた。

## フィナーレ

それから五年後。

山田浩一は、五十歳になっていた。

大和精密工業は、業界トップ企業に成長し、
「価値創造本部」は百名を超える組織になっていた。

山田が支援した企業は、一万社を超えた。
影響を受けた人々は、百万人を超えた。

そして、何より——

「価値起点経営」は、もはや特別なことではなくなっていた。

多くの企業が、当たり前のように実践している。

大学のビジネススクールでも、教えられている。

新しい「標準」になったのだ。

ある日、山田は宮本顧問の訃報を受け取った。

享年八十五歳。

葬儀には、山田が支援した企業の経営者たちが、全国から集まった。

「宮本先生は、私たちに『価値』を教えてくれました」

弔辞で、山田は語った。

「そして、その教えは、今も生き続けています」

「先生が蒔いた種は、大きな森になりました」

「この森は、これからも成長し続けます」

「そして、いつか——」

山田は、集まった人々を見渡した。

「日本中が、世界中が、この森になると信じています」

葬儀の後、山田は一人、宮本の墓前に立った。

「先生、約束を果たせました」

「価値起点の考え方を、多くの人に届けられました」

「でも、これで終わりじゃありません」

山田は、空を見上げた。

青く澄んだ空が、どこまでも広がっている。

「この学びを、次の世代に渡します」

「そして、価値創造が当たり前の社会を作ります」

「必ず」

風が吹いた。

まるで、宮本が「頑張れ」と言っているようだった。

---

山田浩一は、今日も歩き続ける。

疲弊していた四十代の営業マンは、もういない。

今、そこにいるのは——

使命を持って、価値を届け続ける、一人の人間。

そして、彼の旅は、まだ終わらない。

なぜなら——

価値創造に、終わりはないから。

毎日が、新しい朝だから。

毎日が、新しい挑戦だから。

---

**太陽が昇る。**

**新しい朝が始まる。**

**山田浩一の、そして多くの人々の、新しい人生が輝き続ける。**

**それは、終わりではなく、始まり。**

**価値の連鎖は、これからも、永遠に続いていく——**
