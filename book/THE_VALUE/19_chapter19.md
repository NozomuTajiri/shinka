# 第19章 高付加価値の標準化

最高のサービスを誰でも提供できる状態に

---

## 1. 標準化への抵抗

八月の第二月曜日、午後二時。

営業本部長に就任して一ヶ月が経った山田は、全営業部門の部長を集めた会議を開いていた。

「今日は、重要な提案をしたい」

山田は、スクリーンに資料を映し出した。

```
【営業標準化プロジェクト】

目的: 全員が高付加価値を提供できる組織へ
```

しかし、その言葉に、ベテランの西部営業部長が顔をしかめた。

「山田本部長、ちょっと待ってください」

五十代半ばの西部は、営業一筋三十年のベテランだ。

「標準化、ですか......?」

西部の声には、明らかな不快感が含まれていた。

「そうだ。これまで営業部で実践してきた『価値提案の手法』を、全社に展開したい」

「それは......」西部は言葉を選んだ。「営業の質を下げることになりませんか?」

会議室に、緊張した空気が流れた。

---

## 2. 標準化への誤解

「質を下げる?」

山田は、落ち着いて聞き返した。

「はい」西部は腕を組んだ。「営業というのは、属人的なものです。それぞれが長年培ってきた経験、人脈、ノウハウがある」

「それを『標準化』してしまったら、全員が同じことをする。つまり、『平均化』される。優秀な営業マンが、凡庸になってしまう」

西部の意見に、何人かの部長が頷いた。

「確かに......」

「標準化って、マニュアル人間を作ることですよね」

「個性が失われるのでは......」

山田は、部長たちの反応を静かに聞いていた。

これは、予想していた反応だった。

「みんな、『標準化』という言葉を誤解している」

山田は、新しいスライドを映した。

```
【標準化の誤解】

誤解: 標準化 = 平均化 = 質の低下

真実: 標準化 = 最高の標準化 = 質の向上
```

「標準化とは、『下げる』ことじゃない。『上げる』ことだ」

---

## 3. 宮本の教え

山田は、スライドを進めた。

「先日、宮本顧問に相談した。『標準化をどう進めるべきか』と」

そのとき、宮本はこう言った。

**『山田君、標準化とは何だと思う?』**

**『多くの人は、標準化を「普通のレベルに揃える」ことだと勘違いしている。でも、それは違う』**

**『本当の標準化とは、「最高のレベルを、標準にすること」だ』**

山田は、その言葉を反芻するように繰り返した。

「最高のレベルを、標準にする」

西部が、わずかに表情を変えた。

「つまり、こういうことだ」

山田は、営業部の過去半年間のデータを示した。

```
【営業部の成果分布（標準化前）】

トップ20%: 月平均3,000万円受注（高付加価値提案）
中間60%: 月平均1,000万円受注（既存商品提案）
下位20%: 月平均500万円受注（価格競争）

全体平均: 1,200万円
```

「見てほしい。トップ20%の営業は、他の三倍の成果を出している」

「でも、これは『才能』の差じゃない。『やり方』の差なんだ」

山田は、次のデータを示した。

```
【トップ営業の共通点】

1. お客様の本当の課題を聞く（深掘り質問）
2. 六つの価値から最適な提案をする
3. 価値を定量化して伝える
4. お客様の感情に寄り添う
5. 長期的な関係を築く
```

「これらは、すべて『再現可能』なスキルだ」

山田は力を込めて言った。

「標準化の目的は、このトップ20%のやり方を、全員ができるようにすること。つまり——」

スライドが切り替わった。

```
【標準化後の目標】

全員がトップレベルの価値提案ができる組織へ

→ 全体平均を1,200万円から2,500万円へ
```

西部が、目を見開いた。

---

## 4. ベストプラクティスの抽出

「では、具体的に何をするのか」

山田は、プロジェクトの全体像を説明し始めた。

「第一段階は、『ベストプラクティスの抽出』だ」

スライドには、プロセスが示された。

```
【ベストプラクティス抽出プロセス】

1. トップ営業の商談を分析
   └─ 録音、商談記録、提案書を収集

2. 成功パターンを特定
   └─ 何が成功要因だったか

3. 再現可能な形に体系化
   └─ 誰でもできるステップに分解

4. ツール化
   └─ テンプレート、チェックリスト作成
```

「例えば、佐藤課長の『タナカ工業の技能継承提案』」

山田は、具体例を示した。

「この商談では、佐藤は以下のステップを踏んだ」

```
【佐藤課長の成功パターン】

Step 1: 表面的な要求を聞く
   「ベテラン社員の退職後が心配」

Step 2: 深掘り質問
   「具体的に、何が失われますか?」
   「それが失われると、どんな影響がありますか?」
   「本当に守りたいものは何ですか?」

Step 3: 本質的課題の特定
   → 「先代から受け継いだ技術を次世代に残したい」（社会的意義）

Step 4: 価値の定量化
   → 技能継承失敗 = 生産性30%低下 = 年間5,000万円の損失

Step 5: ソリューション提案
   → AIによる暗黙知の形式知化

Step 6: 感情に訴える
   → 「先代からの技術を守れます」
```

「このステップは、誰でも使える」

山田は、部長たちを見渡した。

「才能じゃない。技術なんだ」

---

## 5. 標準化ツールの開発

「第二段階は、『ツールの開発』だ」

山田は、実物を配布した。

それは、A4サイズの冊子だった。

```
【価値提案ガイドブック】

1. 深掘り質問リスト
2. 六つの価値チェックシート
3. 価値定量化テンプレート
4. 提案書フォーマット
5. クロージング会話例
```

「これは、トップ営業の『暗黙知』を『形式知』に変換したものだ」

若手の北部営業部長が、興味深そうに冊子をめくっている。

「例えば、『深掘り質問リスト』を見てほしい」

山田は、ページを開いた。

```
【深掘り質問リスト】

レベル1: 表面的要求を聞く
   「どんな課題がありますか?」

レベル2: 具体化する
   「具体的には、どういうことですか?」
   「それは、どのくらいの頻度で起きますか?」

レベル3: 影響を聞く
   「それが起きると、どんな影響がありますか?」
   「誰が、どう困りますか?」

レベル4: 感情を聞く
   「それについて、どう感じていますか?」
   「理想は、どんな状態ですか?」

レベル5: 本質に迫る
   「本当に大切にしたいことは何ですか?」
   「それが実現すると、どんな価値がありますか?」
```

「この質問を順番に聞いていけば、お客様の本質的な課題にたどり着ける」

西部が、冊子を真剣に読んでいた。

「これは......確かに、使えるかもしれない」

## 6. 研修プログラムの構築

「第三段階は、『研修プログラム』だ」

山田は、研修の全体像を示した。

```
【価値提案研修プログラム】

Phase 1: 基礎研修（2日間）
   - 価値とは何か
   - 六つの価値の理解
   - 深掘り質問の実践

Phase 2: ロールプレイ（1日）
   - 実際の商談を想定
   - フィードバックと改善

Phase 3: OJT（3ヶ月）
   - 実商談で実践
   - 週次レビュー

Phase 4: 認定試験
   - 実商談の録音審査
   - 提案書評価
```

「この研修を通過した人だけが、『価値提案認定営業』として認められる」

「そして、認定された人には——」

山田は、にやりと笑った。

「給与テーブルを一段階上げる」

どよめきが起きた。

「本部長、それは本気ですか?」

「本気だ。専務と社長の承認も得ている」

山田は頷いた。

「高付加価値を提供できる人材には、それに見合う報酬を払う。それが、本当の評価だ」

西部が、深く頷いた。

「なるほど......これは、やる気になる」

## 7. 実践と検証

「そして、最終段階が『実践と検証』だ」

山田は、KPIを示した。

```
【標準化の成功指標】

1. 営業スキル
   - 認定営業の比率: 80%以上
   - 平均質問深度: レベル4以上

2. 商談成果

   - 利益率: 30%以上

3. 顧客満足

   - リピート率: 80%以上

4. 組織文化
   - 社員満足度: 4.0/5.0以上
   - 離職率: 5%以下
```

「これらの指標を、毎月追跡する」

「そして、うまくいかない部分は、すぐに改善する」

山田は続けた。

「標準化は、『固定化』じゃない。『継続的改善』だ」

「トップ営業が新しい手法を見つけたら、それをまた標準に加える。常に、最高のやり方を標準にし続ける」

北部部長が、手を挙げた。

「素晴らしいです。ぜひ、うちの部門からパイロットをやらせてください」

「ありがとう、北部部長」

山田は微笑んだ。

「では、北部部門を第一期として、来月から開始する」

## 8. 三ヶ月後の成果

十一月、第一期研修の成果報告会が開かれた。

北部営業部の二十人が研修を修了し、そのうち十八人が認定試験に合格していた。

「では、成果を報告します」

北部部長が、データを示した。

```
【第一期の成果（3ヶ月間）】

受注額: 前年同期比250%
利益率: 24% → 38%
顧客満足度: 3.2 → 4.5
営業メンバーの満足度: 3.5 → 4.6
```

「特筆すべきは、これまで苦戦していた若手営業の成長です」

北部部長は、一人の若手を紹介した。

「入社二年目の田中です。研修前は、月平均300万円の受注でしたが——」

田中が、恥ずかしそうに立ち上がった。

「先月は、1,500万円を受注しました」

拍手が起きた。

「何が変わったんですか?」

西部部長が聞いた。

「深掘り質問です」

田中は、自分の経験を語り始めた。

「これまで、お客様の言葉をそのまま受け取って、製品を提案していました。でも、研修で学んだ『レベル5までの質問』を使うと——」

田中は目を輝かせた。

「お客様が本当に大切にしていることが、見えてきたんです」

「例えば、『コストを下げたい』と言っていた社長が、本当に心配していたのは、『若手社員の雇用を守ること』でした」

「だから、私は『生産性向上による利益確保で、雇用を守る提案』をしました」

「社長は涙ぐんで、『君は、うちの会社のことを本当に考えてくれている』と言ってくれました」

会議室が、静まり返った。

これが、標準化の力だった。

## 本章のキーメッセージ

**標準化の真実: 最高を標準に**

多くの人が「標準化=平均化=質の低下」と誤解している。

しかし、本当の標準化は、その逆だ。

```
【標準化の3原則】

1. 最高のレベルを標準にする
   └─ トップ20%の手法を全員が使えるように

2. 暗黙知を形式知に変換する
   └─ 属人的スキルを再現可能なツールに

3. 継続的に改善する
   └─ 常に最新のベストプラクティスを標準に
```

標準化によって実現するのは:

- **組織レベルの向上**: 全体の生産性が劇的に上がる
- **個人の成長加速**: 誰もがトップレベルからスタートできる
- **知識の民主化**: 一部のエリートだけでなく、全員が活躍できる

そして、最も重要なのは——

**全員が高付加価値の仕事をすることで、全員が誇りを持って働けること**

これが、標準化の本質である。

山田は、営業部門での成功を全社に展開しようとしている。

技術部門、製造部門、管理部門——すべての部門で、「最高を標準に」する。

これは、組織変革の完成形であり、新しい始まりでもある。
