# 第17章 再現性という武器

仕組みにしなければ、価値は続かない

---

## 1. 成功の落とし穴

八月の第一月曜日。

営業本部長に就任して一ヶ月が経った山田は、新しいオフィスで憂鬱な表情を浮かべていた。

目の前には、先月の業績レポートが広げられている。

「おかしい......」

営業部の受注額は、前月比で横ばい。伸びが止まっていた。

それだけではない。

新しく配属された若手メンバーからは、こんな声が上がっていた。

「山田本部長、お客様の課題を聞いてこいと言われても、何を聞けばいいのかわかりません」

「価値提案をしろと言われても、具体的にどう提案すればいいのか......」

山田は、頭を抱えた。

この一ヶ月、自分は何をしていたのだろう。

営業本部長になって、全社の営業戦略を考えることに追われていた。会議、プレゼンテーション、経営陣への報告——毎日が慌ただしく過ぎていった。

でも、気づかなかった。

自分が実践してきた「価値起点の営業」は、まだ一部のベテランメンバーにしか伝わっていなかった、ということに。

「これは......まずい」

山田は、すぐに宮本顧問に電話をかけた。

---

## 2. 宮本の指摘

翌日、山田は宮本の自宅を訪れた。

「先生、困っています」

山田は、現状を説明した。

営業部の業績が伸び悩んでいること。新人が「価値提案」を実践できていないこと。そして、自分がどうすればいいのかわからないこと。

宮本は、静かに聞いていた。

「山田君、君は大きな間違いを犯している」

「え......？」

「君は、『自分ができること』と、『組織ができること』を混同している」

宮本は、ノートを開いた。

「君は、この半年間で『価値起点の営業』を身につけた。素晴らしいことだ」

「はい......」

「でも、それは『山田浩一』という個人のスキルに過ぎない」

宮本は、ペンを走らせた。

```
【個人の成功 vs 組織の成功】

個人の成功:
- 山田本部長が優秀な営業をする
- 佐藤課長が大型案件を受注する
- ベテランメンバーが活躍する

↓

これは「属人的な成功」

組織の成功:
- 誰がやっても、一定水準の成果が出る
- 新人でも、価値提案ができる
- ベテランの知識が、全員に共有される

↓

これは「仕組み化された成功」
```

山田は、目を見開いた。

「つまり......私のやり方は、仕組みになっていなかった、ということですか？」

「そうだ」宮本は頷いた。「君は、自分で学び、実践し、成果を出した。素晴らしいことだ」

「はい......」

「でも、それを『再現可能な仕組み』にしなければ、組織は成長しない」

宮本は、厳しい目で山田を見た。

「山田君、ビジネスにおいて最も重要なのは何だと思う？」

山田は、少し考えた。

「お客様に価値を届けること......ですか？」

「それもあるが、もっと大事なことがある」

宮本は、大きな文字で書いた。

```
【再現性がなければ、ビジネスではない】
```

「再現性......」

「そうだ。一度だけの成功は、偶然だ。二度目も成功して、初めて『再現性がある』と言える」

宮本は続けた。

「そして、誰がやっても成功する。それが『仕組み化』だ」

山田は、深く頷いた。

「つまり、私がやるべきことは......」

「『山田浩一の営業スキル』を、『誰でも使える仕組み』に変えることだ」

---

## 3. 仕組み化の第一歩

翌週、山田は営業部の全メンバーを集めた。

「今日から、営業プロセスの標準化を始める」

ホワイトボードに、山田は大きく書いた。

```
【営業プロセスの標準化プロジェクト】

目標: 誰でも「価値提案」ができる仕組みを作る
```

若手の松井が、戸惑った表情で聞いた。

「本部長、標準化って......マニュアルを作るということですか？」

「そうだ」山田は頷いた。「でも、ただのマニュアルじゃない」

山田は、スライドを映した。

「これまで、俺たちベテランがやってきたことを分解して、『誰でも実践できるステップ』にする」

画面には、こう書かれていた。

```
【価値提案営業の5ステップ】

Step 1: お客様の課題を発見する
Step 2: 課題の本質を見極める
Step 3: 解決策（価値）を設計する
Step 4: 価値を見える化する
Step 5: 提案する
```

「これが、俺たちの営業プロセスだ」

山田は続けた。

「でも、このままでは抽象的すぎる。だから、各ステップを『具体的な行動』に分解する」

佐藤課長が、手を挙げた。

「本部長、具体的には、どうやって分解するんですか？」

「実際にやってみせる」

山田は、ホワイトボードの前に立った。

「まず、Step 1『お客様の課題を発見する』を分解してみよう」

---

## 4. プロセスの分解

山田は、ホワイトボードに書き始めた。

```
【Step 1: お客様の課題を発見する】

1.1 訪問前の準備
  □ お客様の業界を調べる（業界動向、課題）
  □ お客様の会社を調べる（事業内容、強み、弱み）
  □ 競合他社の動きを調べる

1.2 訪問時のヒアリング
  □ 「最近、どんなことに困っていますか？」と聞く
  □ 「それは、いつから困っていますか？」と聞く
  □ 「それによって、どんな影響がありますか？」と聞く
  □ 「もしそれが解決したら、どうなりますか？」と聞く

1.3 訪問後の分析
  □ ヒアリング内容をノートにまとめる
  □ 「表面的な課題」と「本質的な課題」を分ける
  □ 次回の訪問で確認すべきことをリストアップする
```

若手メンバーたちが、一斉にメモを取り始めた。

「わかりやすい......」

松井が、目を輝かせた。

「これなら、自分にもできそうです」

「そうだろう？」山田は微笑んだ。「ポイントは、『何をすればいいか』が明確になっていることだ」

佐藤課長が、質問した。

「本部長、でも、こんなに細かくチェックリストにしたら、現場の裁量がなくなりませんか？」

「いい質問だ」山田は頷いた。「実は、これは『最低限やるべきこと』のリストだ」

山田は続けた。

「ベテランの佐藤課長なら、この倍以上のことをやっているだろう。でも、新人はどこから手をつければいいかわからない」

「なるほど......」

「だから、まずは『最低限これをやれば、一定の成果が出る』という基準を作る。それが標準化だ」

山田は、次のスライドを映した。

```
【標準化の目的】

× ベテランを標準に合わせる
○ 新人を最低ラインまで引き上げる

× 現場の裁量を奪う
○ 迷わず行動できる指針を与える

× 全員を同じにする
○ 全員が最低限の成果を出せるようにする
```

若手の伊藤が、安堵した表情で言った。

「これなら、私たちでも実践できます」

「そうだ」山田は力強く頷いた。「そして、このプロセスを全員が実践すれば、営業部全体の成果は確実に上がる」

---

## 5. マニュアル作成の実践

その後、山田は営業部を五つのチームに分けた。

各チームが、五つのステップのうち一つを担当し、「具体的な行動マニュアル」を作成することになった。

山田は、各チームにベテランメンバーと新人メンバーを配置した。

「ベテランは、自分がやっていることを言語化してくれ。新人は、わからないことを徹底的に質問してくれ」

「はい!」

一週間後、各チームから報告があった。

まず、「Step 1: 課題発見」チーム。

佐藤課長が代表して発表した。

「私たちは、『課題発見のための質問リスト30』を作りました」

配られた資料には、具体的な質問が並んでいた。

```
【課題発見のための質問リスト30】

＜現状把握の質問＞
1. 現在、どのような業務フローで仕事をされていますか？
2. その業務フローで、一番時間がかかる工程はどこですか？
3. その工程では、誰が、どのくらいの時間をかけていますか？

＜課題の質問＞
4. 最近、困っていることは何ですか？
5. それは、いつから困っていますか？
6. それによって、どんな影響が出ていますか？

＜優先順位の質問＞
7. その課題は、どのくらい緊急ですか？
8. 解決しないと、どうなりますか？
9. 今、一番解決したい課題は何ですか？

（以下、30問続く）
```

「すごい......」

若手メンバーたちが、感嘆の声を上げた。

「これがあれば、訪問時に何を聞けばいいか迷わない」

次に、「Step 3: 価値設計」チーム。

中堅の高橋が発表した。

「私たちは、『六つの価値フレームワーク』を整理しました」

```
【六つの価値フレームワーク】

1. 生産性向上
   └─ 作業時間を△△%削減
   └─ 生産量を△△%増加
   └─ 人件費を△△万円削減

2. リスク回避
   └─ 不良品を△△%削減
   └─ リコールリスクを△△%低減
   └─ 訴訟リスクを回避

3. 時間節約
   └─ 作業時間を△△時間短縮
   └─ 納期を△△日短縮
   └─ 意思決定を△△倍速く

4. 感情的満足
   └─ 従業員満足度を△△%向上
   └─ 顧客満足度を△△%向上
   └─ ストレスを△△%軽減

5. ステータス向上
   └─ 業界シェアを△△%拡大
   └─ ブランド価値を△△%向上
   └─ 競合優位性を確立

6. 社会的意義
   └─ CO2排出量を△△%削減
   └─ 地域雇用を△△人創出
   └─ SDGs目標△△に貢献
```

「これを使えば、どの価値を提案すればいいか、すぐにわかります」

高橋は続けた。

「そして、『△△』の部分に、具体的な数字を入れる。これで、お客様に伝わる価値提案ができます」

山田は、深く頷いた。

「素晴らしい。これこそ、仕組み化だ」

---

## 6. チェックリストの力

さらに一週間後、すべてのチームからマニュアルが提出された。

山田は、それらをまとめて、「価値提案営業マニュアル ver.1.0」を完成させた。

全部で百二十ページ。

営業プロセスの各ステップが、具体的な行動レベルまで分解されていた。

そして、山田は新しい仕組みを導入した。

「今日から、すべての営業活動に『チェックリスト』を使う」

若手の松井に、一枚のシートを渡した。

```
【お客様訪問チェックリスト】

訪問前（30分前までに完了）
□ お客様の業界動向を調べた
□ お客様の会社情報を調べた
□ 前回の訪問議事録を読んだ
□ 今回のヒアリング目的を明確にした
□ 質問リストを準備した

訪問中（必ず実施）
□ 現状の業務フローをヒアリングした
□ 課題を最低3つ聞いた
□ 課題の影響範囲を確認した
□ 優先順位を確認した
□ 次回のアクションを合意した

訪問後（当日中に完了）
□ 議事録を作成した
□ ヒアリング内容を分析した
□ 提案の方向性を決めた
□ 次回訪問の準備リストを作った
□ 上司に報告した
```

「松井、次の訪問から、これを使ってくれ」

「はい!」

松井は、チェックリストを手に、目を輝かせた。

「これがあれば、何をすればいいか迷わない」

「そうだ」山田は頷いた。「そして、このチェックリストを全員が使えば、営業部全体の品質が上がる」

## 7. 成果の見える化

一ヶ月後。

営業部の週次ミーティングで、驚くべき報告があった。

若手の松井が、立ち上がった。

「今週、私は初めて『価値提案』で受注しました」

会議室に、拍手が起きた。

「担当している小林製作所から、二千万円の契約をいただきました」

「おお!」

ベテランの佐藤課長が、感心した表情で聞いている。

「マニュアルとチェックリストを使って、お客様の課題を丁寧にヒアリングしました」

松井は、報告を続けた。

「小林製作所の課題は、『ベテラン検査員の負担が大きい』ことでした。そこで、『時間節約』と『リスク回避』の価値を提案しました」

「具体的には？」

山田が聞いた。

「AI画像検査システムです。これによって、検査時間を六十%削減し、不良品の見逃しリスクもゼロにできると提案しました」

松井は、提案書を見せた。

そこには、価値が明確に書かれていた。

```
【ご提案する価値】

1. 時間節約
   - 検査時間を60%削減（月間240時間→96時間）
   - 検査員の残業をゼロに

2. リスク回避
   - 不良品流出リスクを100%削減
   - 品質クレームによる損失を防止
   - お客様からの信頼を守る

3. 投資対効果
   - 初期投資: 2,000万円
   - 年間削減コスト: 800万円
   - 投資回収期間: 2.5年
```

「素晴らしい」

山田は、心から賞賛した。

「松井、お前は一ヶ月前とは別人だ」

「はい」松井は、少し照れくさそうに言った。「マニュアルとチェックリストのおかげです」

「いや」山田は首を振った。「それを実践したのは、お前だ」

山田は、ホワイトボードに書いた。

```
【仕組みの効果】

一ヶ月前:
- 若手は何をすればいいかわからない
- 成果が出ない
- 自信を失う

今:
- マニュアルとチェックリストがある
- 何をすればいいか明確
- 成果が出る
- 自信がつく
```

「これが、仕組み化の力だ」

## 8. 宮本の評価

その週末、山田は宮本を訪ねた。

「先生、報告があります」

山田は、営業プロセスの標準化プロジェクトについて説明した。

マニュアル作成。チェックリストの導入。そして、若手メンバーの成長。

宮本は、満足そうに頷いた。

「山田君、素晴らしい」

「ありがとうございます」

「君は、『個人の成功』を『組織の成功』に変えた」

宮本は、新しい紅茶を淹れながら続けた。

「ビジネスの世界では、『優秀な個人』よりも『優秀な仕組み』の方が価値がある」

「と、言いますと？」

「優秀な個人は、いつか辞める。病気になる。異動する。でも、優秀な仕組みは、永続する」

宮本は、ノートに書いた。

```
【再現性という武器】

属人的な成功:
- 山田本部長がいれば、うまくいく
- 山田本部長がいなければ、ダメになる

仕組み化された成功:
- 誰がやっても、一定の成果が出る
- 人が変わっても、組織は成長し続ける
```

「そうか......」

山田は、深く頷いた。

「つまり、私がやるべきことは、『自分がいなくても回る仕組み』を作ることなんですね」

「その通りだ」

宮本は、優しく微笑んだ。

「山田君、君はこの半年で、二つの大きなことを成し遂げた」

「二つ......ですか？」

「一つ目は、『価値とは何か』を理解し、実践したこと」

「はい......」

「二つ目は、それを『仕組み』にしたこと」

宮本は続けた。

「一つ目だけなら、優秀な営業マンだ。でも、二つ目ができたから、君は経営者になれる」

「経営者......」

「そうだ。経営者の仕事は、『優秀な個人』になることじゃない。『優秀な組織』を作ることだ」

山田は、胸が熱くなった。

「先生、ありがとうございます」

「まだ、終わりじゃない」

宮本は、厳しい目で言った。

「次は、この仕組みを全社に広げろ。技術部門、製造部門、管理部門——すべてに」

「はい。必ずやります」

## 9. 全社への展開

九月の第一週、山田は全社プロジェクトを立ち上げた。

「『価値創造プロセスの標準化』プロジェクト」

技術部長の木村、製造部長の佐々木、そして管理部門の責任者たちが集まった。

「今日から、営業部でやったことを、全社に展開します」

山田は、プロジェクトの目的を説明した。

```
【プロジェクトの目的】

全部門で「お客様に価値を届ける仕組み」を作る

技術部門:
- お客様の課題を解決する製品開発プロセス

製造部門:
- お客様が求める品質・納期・コストを実現する生産プロセス

管理部門:
- 現場が価値創造に集中できる支援プロセス
```

木村部長が、質問した。

「山田さん、技術部門でも、営業と同じようなマニュアルを作るんですか？」

「はい」山田は頷いた。「例えば、『お客様の課題を技術仕様に落とし込むプロセス』を標準化します」

「なるほど......」

佐々木部長も質問した。

「製造部門では、どうなりますか？」

「例えば、『お客様の要求納期を守るための生産計画プロセス』を標準化します」

山田は続けた。

「大事なのは、すべての仕事が『お客様に価値を届ける』という目的で繋がることです」

三ヶ月後、全社で百を超えるマニュアルとチェックリストが完成した。

そして、驚くべき成果が出始めた。

技術部門では、「お客様要求の仕様化プロセス」が標準化され、開発期間が三十%短縮された。

製造部門では、「品質・納期・コスト最適化プロセス」が標準化され、納期遵守率が九十五%から九十九%に向上した。

管理部門では、「現場支援プロセス」が標準化され、現場からの満足度が大幅に向上した。

## 10. 仕組みの真価

十一月の経営会議。

社長の前で、山田は報告した。

「この半年間の成果を報告します」

スクリーンに、グラフが映し出された。

```
【大和精密工業の変化】

受注額:
- 対前年比: +65%

利益率:
- 対前年比: +45%

納期遵守率:
- 95% → 99%

顧客満足度:
- 78点 → 92点

従業員満足度:
- 65点 → 85点
```

社長が、感嘆の声を上げた。

「すごい......山田君、どうやったんだ？」

## 本章のキーメッセージ

**再現性がなければ、ビジネスではない**

山田は、営業本部長に昇進した後、重要な問題に直面した。

自分が実践してきた「価値起点の営業」は、一部のベテランメンバーには伝わっていたが、新人には伝わっていなかった。

なぜなら、それが「仕組み」になっていなかったからだ。

宮本顧問は、山田に厳しく指摘した。

「再現性がなければ、ビジネスではない」

個人の成功は、その人がいる間だけのものだ。

でも、仕組み化された成功は、組織の未来を創る。

```
【再現性という武器】

属人的な成功:
- 優秀な個人に依存
- その人がいなくなれば終わり
- スケールしない

仕組み化された成功:
- 誰でも実践できる
- 人が変わっても続く
- 組織全体にスケールする
```

山田は、営業プロセスを徹底的に分解し、マニュアルとチェックリストに落とし込んだ。

これによって、新人でも「価値提案」ができるようになった。

さらに、この仕組みを全社に展開し、技術部門、製造部門、管理部門のすべてが「お客様に価値を届ける仕組み」を持つようになった。

結果、大和精密工業は、受注額+65%、利益率+45%という驚異的な成長を遂げた。

そして、最も重要なことは——

この成果が、「山田浩一」という個人に依存しなくなったことだ。

仕組みがあれば、誰でも価値を創造できる。

これが、再現性という武器の真価である。

山田は、取締役に昇進した。

でも、彼の本当の成果は、役職ではない。

「永続する価値の仕組み」を創ったことだ。

この仕組みは、十年後、二十年後の大和精密工業を支え続ける。

それこそが、真の経営者の仕事である。
